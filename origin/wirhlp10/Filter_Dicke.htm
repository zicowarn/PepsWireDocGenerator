Filter Dicke

Um in der Datenbank nach einer bestimmten Materialdicke zu suchen oder eine neue Materialdicke einzutragen, aktiviert man zunächst die Checkbox Filter Dicke.

Dies schaltet die daneben liegende Combobox frei, so dass man diese öffnen kann. Das System zeigt in der Combobox alle in der Datenbank gespeicherten Materialdicken.

Wählen Sie nun die Materialdicke nach der gesucht werden soll oder geben Sie eine neue Materialdicke ein.

Klicken Sie jetzt auf die Schaltfläche Liste neu aufbauen um die gesuchte Materialdicke aufzulisten oder verwenden Sie die Schaltfläche Technologie speichern um eine neu eingegebene Materialdicke zu speichern.
