Zylindrischen Schnitt programmieren

  • Zeichnen Sie die Bearbeitungsfigur(en) die bearbeitet werden soll.

  • Füllen Sie im Menü Bearbeitung den Dialog Operation starten aus.

  • Rufen Sie im Menü Bearbeitung den Befehl Standardkonturbearbeitung auf.

  • Wählen Sie die zu bearbeitenden Figur(en) mit der Schaltfläche .

  • Klicken Sie im Feld Schnittparameter auf die Schaltfläche Einstellungen. Dies ruft den Dialog zur Einstellung der Bearbeitung auf.

  • Aktivieren Sie die Checkboxen unterhalb des Feldes Schnitt Nr., um die Anzahl der Schnitte festzulegen.

  • Geben Sie im Feld Offset das Schlichtaufmaß. Das Aufmaß wird normalerweise für die Simulation verwendet.

  • Bestimmen Sie mit den Schaltflächen die Technologie für den(ie) Schnitt(e).

  • Klicken Sie auf das Register Anbinden.

  • Aktivieren Sie die Option Anbindungslänge und geben Sie im Textfeld die Breite der Anbindung ein, mit der das Ausfallteil gesichert werden soll.

  • Wählen Sie im Bereich Methode zur Entfernung der Anbindung z.B. die Methode Automatisches Entfernen der Anbindung am Ende der Operation.

  • Klicken Sie auf das Register Ein-/Ausfahren.

  • Stellen Sie ein, wie die Bearbeitungsgeometrie(n) an-, bzw. abgefahren werden soll(en).

  • Klicken Sie auf das Register Offset.

  • Aktivieren Sie Option Richtung automatisch (Nur geschl. Figuren) und stellen Sie ein, in welcher Richtung (Uhrzeiger- / Gegenuhrzeiger) die Bearbeitung ausgeführt werden soll.

  • Schließen Sie alle Dialogbe mit OK.

  • Die Bearbeitungsoperation wird im Operationsmanager gespeichert.

  • Führen Sie eine Simulation der Bearbeitung durch.
