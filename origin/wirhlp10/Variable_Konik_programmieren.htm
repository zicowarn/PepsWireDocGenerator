Variable Konik programmieren

Variable Konik programmieren (Zwei Möglichkeiten).

1.) Bestimmen Sie zunächst eine umlaufend gleiche Konik für das gesamte Teil.

Anschließend wird mit der Option Variable Konik an den Geometrieelementen, an denen die Konik von der umlaufenden Konik abweicht, eine andere Konik zugeordnet.

 

2.) Alternativ wird zunächst ein Konikwinkel von 0 Grad im Feld Winkel festgelegt und anschließend die Schaltfläche Variable Konik gewählt, um die Elemente zu bestimmen, an denen sich die Konik ändert.

Nach Wahl der Schaltfläche Variable Konik wird der eingegebene Winkel der gesamten Bearbeitungsfigur zugeordnet und es folgt eine Systemabfrage:

Wähle Position in der Nähe des Elements :

  • Bestimmen Sie nun eine Position in der Nähe der Mitte des Geometrieelementes, auf dem die Konik geändert werden soll.

  • Verwenden Sie die Fangfunktionen Zentrum oder Koordinaten, um die Position zu bestimmen.

Nach Wahl des Elementes wird die Mitte des gewählten Elementes durch ein kleines Kreuz gekennzeichnet und der Dialog Variable Konik aufgerufen.

 

Klicken Sie auf die einzelnen Bereiche des folgenden Bildes, um Hilfetexte aufzurufen.

 

 

Das System zeigt auf der rechten Maustaste oder in der horizontalen Ikonenleiste weitere Optionen:

 

Klicken Sie auf die einzelnen Bereiche des folgenden Bildes, um Hilfetexte aufzurufen.

 

  • Klicken Sie auf eine der Grafiken im Feld Winkelrichtung, um die Konikrichtung einzustellen.

  • Geben Sie jetzt den Endwinkel (neuer Winkel) in den Dialog ein. Wählen Sie anschließend den benötigten Eckenmodus.

  • Die Schaltfläche OK führt die Eingaben aus und fügt das Winkelwechselsymbol (roter Kreis) in die Grafik ein.

Das System positioniert das Winkelwechselsymbol automatisch wird in die Mitte des gewählten Elementes.

Das Symbol (und die zugeordnete Konik) kann mit der Option Löschen entfernt werden.

  • Wiederholen Sie den Vorgang solange, bis alle Konikwechsel definiert sind.

Bei Ausführung des Befehls stehen die Optionen Kopieren + Verschieben, Kopieren + Rotieren und Spiegeln (siehe rechte Maustaste oder Horizontale Ikonenleiste) zur Verfügung.

Die Option dienen zum Vervielfältigen von Winkelwechselsymbolen.
