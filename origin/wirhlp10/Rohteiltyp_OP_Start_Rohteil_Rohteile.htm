Rohteiltyp

 

Rechteckig

Definition eines rechteckiges Rohteil. Das Rechteck wird durch die Eingabe der linken unteren (XY-Minimum) und rechten oberen Rohteilecke (XY-Maximum) festgelegt.

 

So wird es gemacht:

Rechteckiges Rohteil definieren

 

 

Mit dieser Funktion können Sie im Grafikfenster einen Solid-Body anklicken und davon ein rechteckiges Rohteil ableiten.

Das System legt intern eine unsichtbare, rechteckige Box um den gewählten Solid-Body und erzeugt daraus die Rohteilgeometrie.

 

Kreisförmig

Kreisförmiges Rohteil durch Eingabe von Kreismittelpunkt und Durchmesser.

 

Aus Linien/Bögen

Definition eines Rohteils beliebiger Form, auf der Basis von vorhandenen Linien-/Bogenelementen.

Rohteil aus Linien/Bögen erzeugen

  • Die Form des Rohteils muss als in sich geschlossene Konstruktion (Linien-/Bogenelemente) vorliegen.

  • Klicken Sie anschließend im Dialog Neues Rohteil auf die Schaltfläche Wähle Linien/Bögen >>.

  • Wählen Sie nun die Linien-/Bogenelemente einzeln aus oder mit Hilfe der Option Verbinden (rechte Maustaste o. horizontale Ikonenleiste).

Aus Figur

Definition eines Rohteils beliebiger Form, auf der Basis einer vorhandenen Geometrie vom Typ Figur.

 

Als Solid erzeugen

Ist diese Option aktiv, wird das definierte Rohteil wird nicht als Drahtmodell erzeugt, sondern als durchsichtiger Solid-Body.
