Beispiel: Taschenerodieren programmieren

  • Zeichnen Sie die Bearbeitungsfigur(en) die ausgeräumt werden soll.

  • Füllen Sie im Menü Bearbeitung den Dialog Operation starten aus.

  • Rufen Sie im Menü Bearbeitung den Befehl Standardkonturbearbeitung auf.

  • Wählen Sie die zu bearbeitenden Figur(en) mit der Schaltfläche .

  • Klicken Sie auf das Register Taschenerodieren.

  • Aktivieren Sie die Checkbox Verwende rundes Startloch und geben Sie Startlochdurchmesser im Feld Durchmesser ein.

  • Aktivieren Sie die Checkboxen Zuerst ausräumen und Standard Methode.

  • Bestimmen Sie den Abstand der Mittelpunktsbahnen

  • Klicken Sie im Feld Schnittparameter auf die Schaltfläche Einstellungen. Dies ruft den Dialog zur Einstellung der Bearbeitung auf.

  • Aktivieren Sie die Checkboxen unterhalb des Feldes Schnitt Nr., um die Anzahl der Schnitte festzulegen.

  • Geben Sie im Feld Offset das Schlichtaufmaß für den Ausräumvorgang (Schnitt 1) ein. Das Aufmaß wird für spätere Nachschnitte benötigt.

  • Bestimmen Sie mit den Schaltflächen die Technologie für den Ausräumschnitt und ggf. für die Nachschnitte.

  • Schließen Sie alle Dialoge.

  • Die Bearbeitungsoperation wird im Operationsmanager gespeichert.

 

Beispiel:

Die folgende Zeichnung zeigt eine Taschenerodieroperation.

Es wurde eine 2 mm Startlochbohrung, ein Schlichtaufmaß von 0.23 mm und eine Drahtbahnüberlappung von 0.125 mm (halber Drahtdurchmesser) programmiert.

 

 

Das Schlichtaufmaß wird normalerweise mit dem zweiten Schnitt entfernt. Die Größe des Schlichtaufmaßes gibt man im Feld Offset des Registers Schnitte ein.

Die Technologiewerte stellt man ebenfalls in dem Register Schnitte, unterhalb des Feldes Tech. ein.
