Links / Rechts / Kein Offset

Die Option Links / Rechts werden eingesetzt, wenn "offene" Bearbeitungsfiguren geschnitten werden sollen. Wird Links oder Rechts gewählt, entscheidet der Anwender über die Ausgabe des Offsets.

Basis der Angabe ist die Figurrichtung. Ist die Figurrichtung im Uhrzeigersinn und soll eine Matrize im Uhrzeigersinn geschnitten werden, so ist das Offset rechts.

Das System prüft nicht die Richtigkeit der Angaben.

 

Wird Kein Offset gewählt, berechnet das System die Drahtmittelpunktsbahn ohne Offsetverrechnung und schneidet "auf" der Figur.

 

Hinweis: Mit dem Befehl Figurrichtung zeigen / ändern im Menü Info/Bemaßung können Sie die Richtung von Bearbeitungsfiguren prüfen und ggf. umkehren.
