Aussenkontur (Stempel)

Diese Option wird automatisch freigeschaltet, wenn Sie die Option Richtung automatisch wählen.

Sie stellt automatisch sicher, dass die Richtung einer Bearbeitungsfigur korrekt ist, so dass eine Aussenkontur (Stempel) immer mit linkem oder rechtem Offset geschnitten wird.

 

 

Für Offset links muss die Bearbeitungsfigur im Uhrzeigersinn und für Offset rechts im Gegenuhrzeigersinn definiert sein.

Die Option ermittelt und ändert ggf. die Richtung von Bearbeitungsfiguren automatisch.
