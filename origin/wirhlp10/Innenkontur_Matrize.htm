Innenkontur (Matrize)

Diese Option wird automatisch aktiviert wenn Sie Option Richtung automatisch wählen.

Sie stellt automatisch sicher, dass die Richtung einer Bearbeitungsfigur korrekt ist, so dass eine Innenkontur (Matrize) immer mit linkem oder rechtem Offset geschnitten wird.

 

 

Für Offset links muss die Bearbeitungsfigur im Gegenuhrzeigersinn und für Offset rechts im Uhrzeigersinn definiert sein.

Die Option ermittelt und ändert ggf. die Richtung von Bearbeitungsfiguren automatisch.
