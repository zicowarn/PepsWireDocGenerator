Beispiel: Schneidreihenfolge einer Operation ändern, die eine große Anzahl von Figuren bearbeitet.

  • Editieren Sie eine vorhandene Bearbeitunsoperation mit Hilfe des Operationsmanagers. Die Operation sollte wie im folgenden Beispiel mehr als 3 Figuren enthalten.

  • Klicken Sie auf die Schaltfläche Neu ordnen. Das System zeigt den Dialog Optimierung.

 

Die aktuelle Schneidreihenfolge wird angezeigt durch kleine Pfeilsymbole. Das folgende Bild zeigt, dass die Reihenfolge ziemlich ungeordnet ist.

Die Schneidreihenfolge soll nun so verändert werden, dass diese in der unteren linken Ecke beginnt und reihenweise entlang der X-Achse verläuft.

  • Wählen Sie zu nächst mit Hilfe der Schaltfläche den Startpunkt der ersten zu schneidenden Figur (linke untere Figur).

  • Verschieben Sie anschließend den Schieber im Feld Priorität auf X.

  • Klicken danach auf die Schaltfläche Optimieren.

 

Das folgende Bild zeigt die optimierte Schneidreihenfolge.

  • Schließen Sie den Dialog Optimierung mit der Schaltfläche Schließen.
