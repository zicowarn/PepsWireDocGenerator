Anbinden Register

Dieses Register ruft einen Dialog auf in der man festlegt, wie Werkstück / Ausfallteil angebunden und auf welche Weise die Anbindung (Haltesteg) entfernt werden soll.

Normalerweise befestigt man das Werkstück / Ausfallteil nur mit einer Anbindung. Beachten Sie, dass auch die Definition von Mehrfachanbindungen möglich ist.

Dies geschieht mit dem Befehl Zusätzliche Anbindungen im Menü Vorbereitung.

 

Klicken Sie auf die einzelnen Bereiche des folgenden Bildes, um Hilfetexte aufzurufen.
