Beispielprogramm zur variablen Konik

Die folgende Beispielzeichnung zeigt ein symmetrisches Werkstück, dass mit der Funktion variable Konik bearbeitet werden soll.

Die Bearbeitungsfigur wurde im Uhrzeigersinn definiert. Der kleinere Umriss ist die untere Ebene, d.h. die obere Drahtführung muss nach außen auslenken.

Aus diesem Grunde geben wir die Konikwinkel negativ ein (vergl. Beschreibung Standard Konik).

 

Variable Konik programmieren (Beispielprogramm):

 

 

  • Wählen Sie im Menü Bearbeitung den Befehl Standard Konturbearbeitung .

  • Ermitteln Sie mit der Schaltfläche >> den Namen der Bearbeitungsfigur.

  • Geben Sie 0 Grad im Feld Winkel als Startwinkel ein.

  • Wählen Sie im Feld Eckenmodus die Option ISO + Scharf.

  • Klicken Sie auf die Schaltfläche Einstellungen, um den Dialogbox Bearbeitung einstellen aufzurufen:

  • Klicken Sie das Register Anbinden an und definieren Sie einen Steg von 3 mm.

  • Klicken Sie auf das Register Ein/Ausfahren:

  • Bestimmen Sie als Ein-/Ausfahrstrategie eine Gerade.

  • Klicken Sie auf das Register Offset:

  • Aktivieren Sie im Feld Innenkontur (Matrize) die Option Rechts (Uhrzeiger).

  • Schließen Sie den Offset Dialog. Das System wechselt zurück in den Dialog Standard Konturbearbeitung . Klicken Sie dort auf die Schaltfläche Variable Konik.

Da als Startwinkel 0 Grad bestimmt wurde, wird keine Konik in der Bildschirmgrafik gezeigt.

Wähle Position in der Nähe des Elements :

  • Aktivieren Sie die Fangfunktion Koordinaten und klicken Sie mit der Maus auf die 45 Grad Linie nach dem Radius 1 mm (Siehe Position 1 in Zeichnung).

  • Die gewählte Linie ist die Linie, auf welcher die Konik auf 2 Grad wechseln soll.

  • Klicken Sie im Dialog auf das Bild und bestimmen Sie 2 Grad als Endwinkel. Der Eckenmodus ist ISO + Scharf.

Die Bildschirmgrafik verändert sich und es wird die programmierte Konik gezeigt.

Wähle Position in der Nähe des Elements :

  • Digitalisieren Sie nun einen Punkt in der Mitte des Radius 10 mm, auf der rechten unteren Seite (Siehe Position 2 in Zeichnung).

 

 

  • Bestimmen Sie im Dialog 6 Grad als Endwinkel. Der Eckenmodus bleibt ISO + Scharf.

  • Beachten Sie, wie sich die Konik in der Bildschirmgrafik verändert.

Da unsere Beispielgeometrie symmetrisch ist, bietet sich die Möglichkeit die Winkelwechselsymbole zu spiegeln.

Wähle Position in der Nähe des Elements :

  • Wählen Sie die Option Spiegeln (siehe rechte Maustaste oder  horizontale Ikonenleiste).

Wähle Objekte o. S=Wähle D=Entferne ...................... :

  • Vergrößern Sie falls notwendig und wählen Sie die zwei Winkelwechselsymbole mit der Maus.

  • Beenden Sie mit der Option Ende (siehe rechte Maustaste oder horizontale Ikonenleiste) den Auswahlmodus.

Eingabe erster Punkt der Spiegelachse :

  • Fangen Sie mit der Fangfunktion Schnittpunkt eine Schnittpunkt auf der vertikalen C-Linie die durch X0 verläuft (Siehe Zeichnung).

Eingabe zweiter Punkt der Spiegelachse :

  • Fangen Sie einen anderen Schnittpunkt auf der vertikalen C-Linie, die durch X0 verläuft (Siehe Zeichnung).

Es folgt ein Dialog mit der Frage Soll das Original vorhanden bleiben ?. Klicken Sie auf die Schaltfläche Ja.

  • Beenden Sie den Befehl mit der Option Ende (rechte Maustaste oder horizontale Ikonenleiste)

Die Befehlseingabe für die variable Konik ist nun beendet. Die Bildschirmanzeige sollte aussehen wie die Beispielzeichnung auf oben.

Beachten Sie, dass während der Programmierung die Winkelwechselpunkte als kleine rote Kreissymbole dargestellt werden.

Fehlerhafte Eingaben werden durch Löschen des Winkelwechselsymbol aufgehoben. Verwenden Sie dazu die Option Lösche.

Beachten Sie auch, dass die Option Var. Konik editieren (rechte Maustaste oder horizontale Ikonenleiste) eine Korrektur von fehlerhaften Start/Endwinkel Angaben oder ein Verändern der Eckenart erlaubt.
