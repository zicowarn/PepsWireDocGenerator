Tech. ... (Technologie zuweisen)

 

Klicken Sie auf die einzelnen Bereiche des folgenden Bildes, um Hilfetexte aufzurufen.

 

 

Die Schaltfläche ruft einen Dialog auf, in der Sie jedem Schnitt Technologiewerte zuweisen können.

Beachten Sie, dass Sie die Technologiewerte mit der Option Extra dort Technologie im Dialog, überschreiben können.

 

 

Die Optionen im Dialog können je nach verwendeter NC-Maschine unterschiedlich sein. Normalerweise finden Sie hier Einstellungen für Stromstufe, Offset, Spülparameter, etc. .

 

Beispiel:

Die Option Strom ist an den meisten NC-Steuerungen verfügbar. Damit können Sie jedem Schnitt eine separate Stromstufe zuordnen (z.B. 506).

Die Option Offset hingegen, weist jedem Schnitt ein Offsetregisternummer zu. Das heißt, die Eingabe von 0, 1 oder 2 im Dialog entsprächen z.B. D0, D1 oder D2 in der NC-Ausgabe.

Einmal bestimmt, können Sie Technologien in der Technologie Datenbank speichern. Verwenden Sie dazu den Befehl Speichern ... im Feld Technologie Datenbank.
