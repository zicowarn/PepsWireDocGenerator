Startposition

In diesem Dialog legt man die Drahtstartposition für NURBS-Surface Bearbeitungen fest. Soll der Draht senkrecht starten reicht die Eingabe einer XY-Position.

Soll der Draht in einem bestimmten Winkel anfahren, müssen Sie eine zweite XY-Position im Feld UV-Startposition eingeben, die das System dann in UV-Daten umrechnet.
