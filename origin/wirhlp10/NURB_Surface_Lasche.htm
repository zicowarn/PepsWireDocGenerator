NURBS-Surface Register

Dieses Register bietet die Möglichkeit eine Regelflächenbearbeitung aus einer 3D-Fläche von Typ NURBS-Surface zu erzeugen.

Bei einer normalen Regelflächenbearbeitung liegen die untere bzw. obere Ebene einer Regelfläche auf der XY-Ebene, OHNE Abweichung in der Z-Ebene.

Die Option NURBS-Surface schneiden erlaubt die Bearbeitung von 3D-Regelflächen, deren obere bzw. untere Ebene aus einer 3D-NURBS-Kurve besteht. Das heißt, es sind Abweichungen in der Z-Achse vorhanden.

 

 

NURBS-Surfaces können von den meisten externen CAD-Systemen als IGES Datei ausgegeben werden.

Aus diesem Grunde kann man NURBS-Surfaces in PEPS einladen und bearbeiten.

Ist kein CAD-System vorhanden, kann man das NURBS-Surface mit dem Befehl NURBS-Kurven und Surfaces im Menü CAD erzeugen.
