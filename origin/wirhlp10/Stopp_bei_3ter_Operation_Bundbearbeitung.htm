Stopp bei 3ter Operation

Mit dem Dialog Bundbearbeitung können Sie für einige CNC-Maschinen (z.B. ACVision) Bundbearbeitungen programmieren, die aus 3 Bereichen bestehen.

Nämlich aus einer oberen Konik, einem mittleren zylindrischen Bereich und einer unteren Konik bestehen.

 

 

Die Option Stopp bei 3ter Operation definiert, ob für den mittleren, zylindrischen Bereich ein zusätzlicher Stopp (z.B. M01) ausgegeben werden soll.
