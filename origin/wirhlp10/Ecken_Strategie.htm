Ecken Freimachung

Wurde eine Eckenfreimachung programmiert (Siehe Menüfolge Vorbereitung \ Eckenfreimachung), wird diese Freimachung nur dann geschnitten, wenn die Felder unterhalb Ecken Freimach aktiv sind.

Auf diese Weise können Sie z.B. Schnitt 1 mit einer Freimachung und Schnitt 2 / Schnitt 3 ohne Freimachung schneiden.

 

Hinweis:

Eckenfreimachungen müssen vor einer Bearbeitungsprogrammierung mit dem Befehl Eckenfreimachung aus dem Menü Vorbereitung, in die Bearbeitungsfigur eingebracht werden.

 

So wird es gemacht:

Eckenfreimachung bestimmen

Eckenfreimachung nachträglich verändern
