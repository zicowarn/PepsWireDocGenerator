Profil auf einer Helix abwickeln

Die zu schneidende Kontur wird in 2D geschnitten, wobei sich das Rohteil um die Rotationsachse dreht. Die Kontur wird dabei helikal auf einer weichen Kurve abgewickelt.

Das heißt, dass sich die Rotationsachse gleichzeitig mit den Schneiddüsen entlang der Rotationsachse bewegt.

Die Bilder unterhalb zeigen die zylindrische Bearbeitung eines Langlochs, bei gleichzeitiger Rotation der Rotationsachse.

 

 

Wurde die Option Profil auf einer Helix abwickeln aktiviert, schaltet das System folgende Optionen frei:

Rotation, Grad pro mm

Der Parameter Rotation, Grad pro mm definiert die Rotation der Rotationsachse (in Grad) pro jedem geschnittenen Millimeter.

X-Position für Start der Helix

Diese X-Positionskoordinate erlaubt es, die Schneidgeometrie entlang der X-Achse zu verschieben. Die helikale Form startet dann an der definierten X-Position.
