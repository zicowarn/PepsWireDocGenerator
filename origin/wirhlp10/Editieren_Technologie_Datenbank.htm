Editieren (Technologie Datenbank)

Ruft einen weiteren Dialog auf, der zeigt wie viel Schnitte, welcher Offset, welches Offsetregister und welche Stromstufe dem gewählten Material zugeordnet ist.

Hinweis: Der Dialog kann je nach NC-Maschine variieren.

 

 

Das Feld Offset Betrag dient ausschließlich dazu, den Offset bei einer Simulation optisch zu veranschaulichen. Es gibt jedoch eine Ausnahme:

Ist bei Bearbeitung die Option Offset ausgeben aktiv, so werden die Offsetwerte in die NC-Daten ausgegeben.

Das Feld Offset zeigt mit welchem Offsetregister ein Schnitt ausgeführt wird.
