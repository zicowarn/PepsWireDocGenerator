Gruppierte Operationen auf Punkten wiederholen

Mit dieser Funktion können Sie eine gruppierte Operation auf beliebigen XY-Positionen (in der Zeichnung durch C-Punkte gekennzeichnet) wiederholen.

 

 

Gruppenname

Wird verwendet, um den Namen der Gruppe auszuwählen, falls mehrere Gruppen existieren.

  • Aktivieren Sie die Option Verwende Punkte-Figur.

XY-Referenzpunkt

Geben Sie die Position ein, die als Bezugspunkt für das Wiederholen der Gruppierten Operationen dienen soll. Oder verwenden die Schaltfläche , um die Position zu digitalisieren.

 

Wähle Punkte-Figur

Wenn die Punkte-Figur schon von einer vorangegangenen Operation erzeugt wurde, können Sie die Schaltfläche verwenden, um die Punkte-Figur in der Zeichnung auszuwählen.

  • Ansonsten wählen Sie die Schaltfläche .

  • Klicken Sie die C-Punkte an (müssen vorher definiert worden sein), auf denen die Gruppierten Operationen wiederholt werden sollen.

  •  Beenden Sie die Auswahl mit der Option Ende (rechte Maustaste od. horizontale Ikonenleiste).

Das System schreibt die Informationen zurück in den Dialog.

 

 

Falls die Punkte willkürlich ausgewählt wurden, können Sie diese mit der Schaltfläche automatisch sortieren lassen.

 

Zeige/Simuliere diese gruppierten Operationen.

Wenn aktiv, werden bei einer Bearbeitungssimulation alle Gruppenpositionen simuliert. Wenn nicht, wird nur die erste Gruppenoperation simuliert.

  • Schließen Sie den Dialog mit OK.

Der Befehl wird ausgeführt und im Operationsmanager angezeigt.

 

 

 

Simulieren Sie das Programm, um das Ergebnis zu überprüfen.
