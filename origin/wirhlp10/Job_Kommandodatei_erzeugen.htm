Job- / Kommandodatei erzeugen

Dieser Befehl erzeugt für NC-Maschinen vom Typ AGIE 100/123 und CHARMILLES Robofil automatisch eine Job- bzw. Kommandodatei (CMD) und lädt diese in einen Editor oder ruft einen Editor zum manuellen Erzeugen der o.g. Dateien auf.

Nach Aufruf des Befehls fragt das System zunächst:

 

 

Wird der Dialog mit Ja bestätigt, erzeugt das System aus den momentanen Programmdaten eine Job- / Kommandodatei und zeigt diese in einem Editor.

Klickt man auf Nein ruft das System nur einen Editor auf, so dass die Job- / Kommandodatei manuell erzeugt werden kann.
