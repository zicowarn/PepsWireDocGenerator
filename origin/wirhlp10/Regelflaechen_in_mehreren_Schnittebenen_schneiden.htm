Regelflächen in mehreren Schnittebenen schneiden

Schnittebenen werden dann angewendet, wenn die Werkstückhöhe die physikalischen Grenzen der NC-Maschine überschreitet oder die maximal mögliche Drahtauslenkung überschritten wird.

In solchen Fällen wird das Werkstück aus mehreren Schnittplatten, die nacheinander geschnitten werden, zusammengebaut. Jede Schnittplatte wird dabei als separate Schnittebene betrachtet.

Die folgende Zeichnung zeigt ein 60 mm hohes Werkstück das in 3 Schnittplatten(ebenen) aufgeteilt ist.

In dem Beispiel wird angenommen, dass nur die Geometrie der unteren und obere Ebene (d.h. Z0 und Z60) bekannt sind.

 

Regelfläche in mehreren Schnittebenen schneiden:

 

Schnitt 1 bearbeiten

  • Starten Sie eine Operation mit Operation starten

  • Definieren Sie die Referenzhöhe auf 0 und die Sekundärebene auf 20 mm.

  • Bearbeiten Sie die Regelfläche mit:

Höhe untere Ebene = 0 und

    Abstand der Ebenen = 60

  • Im Feld Ebenen Berechnung geben Sie bitte ein:

Starthöhe = 0

Endhöhe = 60 und Anzahl Ebenen = 4

 

Schnitt 2 bearbeiten

  • Starten Sie eine Operation mit Operation starten

  • Definieren Sie die Referenzhöhe auf 0 und die Sekundärebene auf 20 mm.

  • Bearbeiten Sie die Regelfläche mit:

Höhe untere Ebene = -20 und

Abstand der Ebenen = 60

  • Im Feld Ebenen Berechnung geben Sie bitte ein:

Starthöhe = 0

Endhöhe = 60 und Anzahl Ebenen = 4

 

Schnitt 3 bearbeiten

  • Starten Sie eine weitere Operation mit Operation starten

  • Definieren Sie die Referenzhöhe auf 0 und die Sekundärebene auf 20 mm.

  • Bearbeiten Sie die Regelfläche mit:

Höhe untere Ebene = -40 und

Abstand der Ebenen = 60

  • Im Feld Ebenen Berechnung geben Sie bitte ein:

Starthöhe = 0

Endhöhe = 60 und Anzahl Ebenen = 4
