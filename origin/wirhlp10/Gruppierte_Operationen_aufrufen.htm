Gruppierte Operationen aufrufen

Mit dieser Funktion können Sie eine Anzahl von Bearbeitungsoperationen aufrufen, die im Vorfeld zu einer Gruppierten Operation zusammengefasst wurden und diese auf Mustern (Teilkreis oder Rechteckmuster) oder an beliebigen Positionen wiederholen.

 

 

So wird es gemacht:

1.) Mehrere Bearbeitungsoperationen zu einer Gruppierten Operation zusammenfassen

2.) Eine Gruppierte Operation aufrufen und an beliebigen anderen Positionen wiederholen

 

Wichtiger Hinweis:

Bevor Sie eine Gruppierte Operation aufrufen können, müssen diese erst erzeugen.  

Siehe: Eine Gruppierte Operation aufrufen und an beliebigen anderen Positionen wiederholen
