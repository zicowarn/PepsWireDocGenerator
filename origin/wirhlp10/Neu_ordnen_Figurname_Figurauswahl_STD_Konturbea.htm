Neu ordnen

Mit dieser Funktion können Sie die Reihenfolge der Bearbeitung ändern, indem Sie die Liste der zu schneidenden Bearbeitungsfiguren automatisch neu ordnen.

Die Funktion ist nur sinnvoll, wenn die Bearbeitungsreihenfolgen von einer größeren Zahl von Bearbeitungsfiguren verändert werden soll.

 

Siehe auch: Schneidreihenfolge einer Operation ändern, die eine große Anzahl von Figuren bearbeitet.
