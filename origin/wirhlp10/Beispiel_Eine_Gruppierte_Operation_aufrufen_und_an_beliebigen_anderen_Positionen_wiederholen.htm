Eine Gruppierte Operation aufrufen und an beliebigen anderen Positionen wiederholen

Um eine gruppiert Operation aufzurufen, existieren zwei Möglichkeiten:

 

Über das Menü mit der Menüfolge Bearbeitung \ Gruppierte Operationen aufrufen.

Oder über den Operationsmanger durch einen Klick mit der rechten Maustaste auf den Operationsstart und Wahl der Option Gruppierte Operationen aufrufen.

 

 

 

Das System zeigt nun den Dialog Gruppenoperation aufrufen.

  • Legen Sie in diesem Dialog fest, welche Gruppe auf welchen geometrischen Muster, z.B. auf einem Teilkreismuster, wiederholt werden soll.
