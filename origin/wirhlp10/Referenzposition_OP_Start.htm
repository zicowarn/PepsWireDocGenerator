Referenzposition

Dieses Feld bestimmt eine Referenzposition, die das System bei der NC-Ausgabe als X0 Y0 betrachtet.

Wenn Sie in diesem Feld keine Koordinate eingeben, verwendet das System den absoluten Zeichnungsnullpunkt als Bezugspunkt für die NC-Ausgabe.

Soll eine andere Position als der absolute Zeichnungsnullpunkt als X0 Y0 für NC-Ausgabe gelten, gibt man deren Koordinate im Feld Referenzposition ein oder ermittelt diese mit Hilfe der Fangfunktionen.

Die Funktion ist besonders nach einem Import von CAD-Daten nützlich, da sie auf eine Transformation der Geometrie auf den absoluten Nullpunkt des CAM Systems verzichten können, wenn Sie die Referenzposition für die NC-Ausgabe in der Zeichnung abgreifen.
