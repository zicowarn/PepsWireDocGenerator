Rechteckiges Rohteil definieren

  • Rufen Sie im Menü Bearbeitung den Befehl Operation starten auf und klicken Sie im Dialog auf das Register Rohteil.

  • Wählen Sie im Bereich Rohteil die Schaltfläche Neues Rohteil.

  • Aktivieren Sie die Option Rechteckig.

  • Geben Sie im Feld XY-Minimum die Koordinate der linken, unteren Werkstückecke ein und im Feld XY-Maximum die der rechten oberen Ecke.

  • Geben Sie im Feld Höhe die Materialhöhe ein.

  • Schließen Sie alle Dialoge, programmieren Sie die Bearbeitung und führen Sie eine Solid-Simulation durch.
