Solid als Rohteil verwenden

  • Definieren Sie die Solidgeometrie des Rohteils (Solid importieren oder konstruieren).

  • Rufen Sie im Menü Bearbeitung den Befehl Operation starten auf und klicken Sie im Dialog auf das Register Rohteil.

  • Klicken Sie im Dialogbereich Rohteile auf die Schaltfläche >>.

  • Klicken Sie das Solid mit der linken Maustaste an und beenden Sie die Auswahl mit der Option Ende (siehe: rechte Maustaste od. horizontale Ikonenleiste).

  • Schließen Sie den Dialog, programmieren Sie die Bearbeitung und führen Sie eine Solid-Simulation durch.
