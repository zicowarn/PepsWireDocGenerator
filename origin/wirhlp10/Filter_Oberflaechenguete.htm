Filter Oberflächengüte

Um in der Datenbank nach einer bestimmten Oberflächengüte zu suchen oder eine neue Oberflächengüte einzutragen, aktiviert man zunächst die Checkbox Filter Oberflächengüte.

Dies schaltet die daneben liegende Combobox frei, so dass man diese öffnen kann. Das System zeigt in der Combobox alle in der Datenbank gespeicherten Oberflächengüten.

Wählen Sie nun die Oberflächengüte nach der gesucht werden soll oder geben Sie eine neue Oberflächengüte ein.

Klicken Sie jetzt auf die Schaltfläche Liste neu aufbauen um die gesuchte Materialdicke aufzulisten oder verwenden Sie die Schaltfläche Technologie speichern um eine neu eingegebene Oberflächengüte zu speichern.
