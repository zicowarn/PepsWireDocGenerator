Erzeugen einer Bearbeitungsfigur

Fast alle Bearbeitungsoperationen des Moduls Drahterosion basieren auf dem Geometrietyp Bearbeitungsfigur.

Sie sollten unbedingt mit der Erzeugung von Konstruktionen auf der Basis von C-Linien, C-Kreisen und Linien- /Bogenelementen vertraut sein, bevor Sie weiterlesen.

Um eine Konturbearbeitung, Regelflächenbearbeitung oder eine Taschenerodierbearbeitung ausführen zu können, müssen Sie die vorhandenen Linien-/Bogenelemente zu einer Bearbeitungsfigur zusammenfassen.

Dies geschieht durch Anwendung der Befehle Linien/Bögen ==> Figur, Parametrische Figuren und Auto. Matritzen / Stempel im Menü Vorbereitung.

Jede Bearbeitungsfigur muss folgende Einzelheiten beinhalten:

 

- Eine Start(loch)position - definiert als Punkt VOR der Figur.

- Eine Bearbeitungsstartposition - definiert als Punkt AUF der Figur. Wird keine Bearbeitungsstartposition angegeben, verwendet das System die nächste Position AUF der Figur, bezogen auf die Start(loch)position.

- Eine Anzahl von Linien/Bogenelementen - welche die zu schneidenden Bearbeitungsfigur bilden. Der Anwender muss sicherstellen, dass die Geometrie der Bearbeitungsfigur von guter Qualität, d.h. maßgenau ist und kein Lücken aufweist.

 

Auch eine OFFENE Bearbeitungsfigur muss eine Start(loch)position und eine Bearbeitungsstartposition aufweisen.

Bei einer Regelflächenbearbeitung benötigt normalerweise nur die Figur, welche die obere Ebene bildet, eine Startposition.
