NC-Ausgabe für einen Maschinentyp anpassen

  • Wählen Sie im Menü Bearbeitung den Befehl Postprozessor konfigurieren.

  • Welchen Sie im Dialog in das Register NC-Ausgabe Optionen.

  • Aktivieren Sie die Option Automatische Konfiguration ausschalten.

  • Wählen Sie aus der Listbox Steuerungstyp einen passenden Maschinen-/Steuerungstyp.

  • Aktivieren Sie die benötigen Einstelloptionen.

  • Speichern Sie die Einstellungen mit Eingaben speichern.
