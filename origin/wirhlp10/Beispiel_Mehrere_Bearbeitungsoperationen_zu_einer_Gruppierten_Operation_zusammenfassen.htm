Mehrere Bearbeitungsoperationen zu einer Gruppierten Operation zusammenfassen

Mehrere vorhandene Bearbeitungsoperationen sollen zu einer Gruppierten Operation zusammengefasst werden.

  • Klicken Sie im Operationsmanager die Bearbeitungen an, die zur Gruppe gehören sollen. Verwenden Sie zur Mehrfachauswahl die Tasten Strg oder Shift.

 

  • Klicken mit der rechten Maustaste auf eine der gewählten Operationen, um das Kontextmenü anzuzeigen.

  

  • Wählen Sie im Kontextmenü die Option Gruppieren. Das System zeigt den Dialog Neue Gruppenoperation erzeugen.

  • Geben Sie einen Namen für die Gruppe ein und bestätigen Sie mit OK.

  

 

Das System zeigt nun den Dialog Gruppenoperation aufrufen.

In dem Dialog wird festgelegt, welche Gruppe auf welchen geometrischen Muster, z.B. auf einem Teilkreismuster, wiederholt werden soll.

 

Klicken Sie auf die einzelnen Bereiche des folgenden Bildes, um Hilfetexte aufzurufen.

 

 

  • Wählen Sie die Option Gruppe später verwenden.
