Passwort ändern

Mit der Option Passwort ändern können Sie das momentane Passwort verändern und Ihr eigenes Passwort vergeben.

Zu diesem Zweck wird zunächst auf die Schaltfläche Passwort ändern geklickt und anschließend im Feld Neues Passwort das neue Passwort eingegeben.

Zur Bestätigung muss anschließend im Feld Passwort bestätigen das neue Passwort noch einmal eingegeben werden.
