Bundbearbeitung programmieren

  • Zeichnen Sie die Bearbeitungsfigur(en) die bearbeitet werden soll.

  • Rufen im Menü Bearbeitung den Dialog Operation starten auf.

  • Bestimmen Sie im Dialog den NC-Dateinamen und die Startposition.

  • Geben Sie im Feld Referenzhöhe die Z-Lage des Bundes ein (Gilt nicht für Agievision-Maschinen) XXX.

(Beispiel für Materialhöhe 40 und 5 mm Bund oben )

  • Rufen Sie im Menü Bearbeitung den Befehl Bundbearbeitung auf.

  • Wählen Sie die zu bearbeitenden Figur(en) mit der Schaltfläche .

 

  • Legen Sie mit den grafischen Schaltflächen im Bereich Standardkonik fest, ob der Bund oben oder unten sein soll.

  • Anschließend in den Feldern Winkel den Grad der Konik ein (positiver Wert).

 

  • Wählen Sie die Registerlasche Technologie.

  • Klicken Sie rechts neben dem Symbol auf die Schaltfläche Einstellungen.

  • Dies ruft den Dialog zur Einstellung der Bearbeitung des zylindrischen Bereichs auf.

  • Aktivieren Sie die Checkboxen unterhalb des Feldes Schnitt Nr., um die Anzahl der Schnitte festzulegen.

  • Geben Sie im Feld Offset das Schlichtaufmaß ein. Das Aufmaß wird normalerweise für die Simulation verwendet.

  • Bestimmen Sie mit den Schaltflächen die Technologie für den(ie) Schnitt(e).

  • Klicken Sie auf das Register Anbinden.

  • Aktivieren Sie Anbindungslänge und geben Sie im Textfeld die Breite der Anbindung ein, falls es ein Ausfallteil gibt und dieses gesichert werde soll.

  • Geben Sie Keine ein, wenn kein Ausfallteil zu erwarten ist.

  • Wählen Sie im Bereich Methode zur Entfernung der Anbindung z.B. die Methode Stopp  Anbindung mit dem ersten Schnitt entfernen.

  • Klicken Sie auf das Register Ein-/Ausfahren und stellen Sie ein, wie die Bearbeitungsgeometrie(n) an-, bzw. abgefahren werden soll(en).

  • Klicken Sie auf das Register Offset.

  • Aktivieren Sie Option Richtung automatisch (Nur geschl. Figuren) und stellen Sie ein, in welcher Richtung (Uhrzeiger- / Gegenuhrzeiger) die Bearbeitung ausgeführt werden soll.

  • Schließen Sie den Einstellungen Dialog mit OK.

 

Bearbeitungsparameter für die konische Bearbeitung einstellen.

  • Klicken Sie rechts neben dem Symbol auf die Schaltfläche Einstellungen. Dies ruft den Dialog zur Einstellung der Bearbeitung des konischen Bereichs auf.

  • Stellen Sie die Bearbeitungsparameter Anzahl der Schnitte, Technologie, Anbindung etc. für die konische Bearbeitung ein. Gehen Sie vor, wie oberhalb beschrieben.

  • Schließen Sie den Einstellungen Dialog mit OK.

  • Wählen Sie die Registerlasche Optionen.

  • Stellen Sie unter Bearbeitungsoptionen ein, mit welcher Strategie die Bundbearbeitung ausgeführt werden soll.

  • Schließen Sie alle Dialoge mit OK.

  • Die Bearbeitungsoperation wird im Operationsmanager gespeichert.

 

  • Führen Sie eine Simulation der Bearbeitung durch.
