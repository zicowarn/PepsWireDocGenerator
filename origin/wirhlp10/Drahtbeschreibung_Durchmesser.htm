Drahtbeschreibung / Durchmesser

Mit diesem Feld können Sie eine neue Drahtsorte zur Liste hinzufügen.

Drahtsorte hinzufügen

  • Rufen Sie im Menü Bearbeitung den Befehl Operation starten auf und wählen Sie das Register Technologie.

  • Klicken Sie im Dialog auf die Schaltfläche .

  • Geben Sie im Feld Beschreibung den Namen der Drahtsorte z.B. Cobra Cut 0.1 ein.

  • Bestimmen Sie im Feld Drahtdurchmesser den Durchmesser des Schneiddrahtes und klicken Sie die Schaltfläche Hinzufügen.
