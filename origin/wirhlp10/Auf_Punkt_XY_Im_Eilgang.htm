Auf Punkt (XY)

In dieses Feld wird die XY-Position notiert, auf die der Draht im Eilgang verfahren soll. Die XY-Position wird direkt in das Feld eingegeben.

Es werden die absoluten X- und Y-Koordinaten der Zielposition angegeben.

Alternativ kann die XY-Zielposition mit der Schaltfläche XY >> aus einer vorhandenen CAD-Zeichnung ermittelt werden.

Verwenden Sie in diesem Fall die Fangfunktionen Endpunkt - Koordinaten.
