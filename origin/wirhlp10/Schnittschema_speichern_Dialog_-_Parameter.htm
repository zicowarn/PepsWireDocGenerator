Schnittschema speichern Dialog - Parameter

Verwenden Sie diese Schaltfläche , um die aktuellen Dialogeinstellungen als neues Schnittschema zu speichern.

Bestehende Schnittschemen können Sie verändern, indem Sie mit Schaltfläche den Schnittschema laden Dialog aufrufen.
Dann das Schnittsschema suchen es anpassen und danach speichern.

Siehe auch: Erodierbearbeitungen durch Schnittschemen automatisieren

 

 

KNO Dateiname

In diesem Feld gibt das System einen Name für das Schnittschema vor. Die Vorgabe besteht aus dem aktuell gewähltem Material, Materialhöhe, Draht, Drahtdurchmesser und der Anzahl der Schnitte.

Beachten Sie, dass man den vorgegeben Name verändern kann. Die Systemvorgabe lässt sich einstellen im Dialog Modul Einstellungen (Menüfolge: Datei > Einstellungen > Modul Einstellungen > Dialogeinstellungen.

 

Schnittschemenfarbe

In einigen Anwendungsfällen sind bestimmte Flächen von 3D Modellen farbig eingefärbt.

So könnte beispielsweise, eine rote Bohrung eine Passung H7 kennzeichnen.

Das Feld Farbe zeigt an, welche Farbe einem Schnittschema zugewiesen wurde.

Der benötigte Farbcode lässt sich mit der Schaltfläche Face picken von einem 3D Modell ableiten oder mit der Schaltfläche Farbe wählen über den MS-Windows Dialog zur Farbauswahl bestimmen.

 

Draht, Draht-Durchmesser

Bestimmt den verwendeten Draht und den Drahtdurchmesser.

Werkstückmaterial, Höhe

Bestimmt das verwendete Material und dessen Höhe.

Durchmesser

XXX Spezialfall - Beschreibung in Arbeit.

Anzahl der Schnitte

Zeigt die im Schnittschema gespeicherte Anzahl von Schnitten.

Beschreibung, Kommentar

Zwei frei beschreibbare Felder, in denen der Anwender einem Schnittschema kommentierende Texte zuweisen kann.
