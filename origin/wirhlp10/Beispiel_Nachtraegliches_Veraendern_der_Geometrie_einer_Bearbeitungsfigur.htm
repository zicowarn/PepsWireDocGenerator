Nachträgliches Verändern der Geometrie einer Bearbeitungsfigur

  • Führen Sie einen Doppelklick auf die Figur aus. (Alternativ können Sie im Menü Layer/Figuren den Befehl Figur editieren aufrufen und dann die Figur anklicken, die es zu verändern gilt.)

  • Das System zeigt einen Dialog (nur bei Doppelklick auf Figur).

  • Klicken Sie auf die Schaltfläche Figur editieren.

  • Verändern Sie die Geometrie mit Hilfe der Konstruktionsbefehle.

  • Führen Sie einen Doppelklick auf das Feld (unten rechts im Grafikfenster) aus oder rufen Sie im Menü Layer/Figuren den Befehl Figur editieren Ende auf.

oder:

  • Lösen Sie die Bearbeitungsfigur mit dem Befehl Figur auflösen im Menü Layer/Figuren auf.

  • Verändern Sie die Geometrie mit Hilfe der Konstruktionsbefehle.

  • Erzeugen Sie die Bearbeitungsfigur erneut.
