Spannmittel für Solidsimulation aus Solidgeometrien erzeugen

  • Definieren Sie die Solidgeometrien der Spannmittel (Solid importieren oder konstruieren).

  • Rufen Sie im Menü Bearbeitung den Befehl Operation starten auf und klicken Sie im Dialog auf das Register Rohteil.

  • Klicken Sie im Dialogbereich Spannmittel auf die Schaltfläche >>.

  • Klicken Sie die Solids mit der linken Maustaste an und beenden Sie die Auswahl mit der Option Ende (siehe: rechte Maustaste od. horizontale Ikonenleiste).

  • Schließen Sie den Dialog, programmieren Sie die Bearbeitung und führen Sie eine Solid-Simulation durch.
