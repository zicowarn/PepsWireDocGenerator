Profil für einen Maschinentyp anlegen

  • Wählen Sie im Menü Bearbeitung den Befehl Postprozessor konfigurieren.

  • Wechseln Sie im Dialog in das Register NC-Ausgabe Optionen.

  • Aktivieren Sie die Option Automatische Konfiguration ausschalten.

  • Falls vorhanden, wählen Sie aus der Listbox Steuerungstyp einen passenden Maschinen-/Steuerungstyp.

  • Aktivieren Sie die benötigen Einstelloptionen.

  • Speichern Sie die Einstellungen mit Eingaben speichern.

  • Wechseln Sie im Dialog in das Register Generell (Allgemein) .

  • Klicken Sie im Bereich Vorhandene Maschinenprofile auf die Schaltfläche . Dies fügt das neue Profil zur Liste der Profile hinzu.

  • Speichern Sie die Einstellungen mit Eingaben speichern.
