Auf Punkt (UV)

In dieses Feld wird die UV-Position notiert, auf welche der Draht im Eilgang verfahren soll.

Die UV-Angabe ist nur dann erforderlich, wenn eine schräge Eilgangbewegung (schräge Drahtstellung) erfolgen soll.

Die UV-Position wird direkt in das Feld eingegeben. Es werden die inkrementalen U- und V-Koordinaten der Zielposition angegeben.

Alternativ kann die UV-Zielposition mit der Schaltfläche UV >> aus einer vorhandenen CAD-Zeichnung ermittelt werden.

Verwenden Sie in diesem Fall die Fangfunktionen Endpunkt - Koordinaten.

Der Dialog bezieht die inkrementale UV-Positionseingabe auf die momentane XY-Eingabe.

Aus diesem Grund muss die XY-Position vor der UV-Positionsangabe programmiert werden.

Für reine 2 Achsenbewegungen ist KEINE Programmierung der UV-Position notwendig.
