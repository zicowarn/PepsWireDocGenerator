Konik auf Punkt auslenken

Als Systemvoreinstellung wird die Konik kontinuierlich auf einem gewählten Element zugestellt.

Mit der Option Konik auf Punkt auslenken, können Sie die Konik auf einer bestimmbaren Punktposition zustellen.

Die folgende Abbildung zeigt eine variable Konik, bei der die Konik an drei Positionen auf einem Punkt ausgelenkt.
