Auswahl beenden / löschen

Auswahl löschen –

Löscht die aktuelle Flächen-/Kantenauswahl.

 

Auswahl beenden

Führt die aktuelle Flächen-/Kantenauswahl aus. Dabei überprüft WIRE-Expert, ob aus den gewählten Flächen/Kanten ein erodierbares Feature erzeugt werden kann.

Falls nicht, wird im Dialog eine Fehlermeldung angezeigt.

 

Auswahl vervollständigen – Diese Schaltfläche steht nur zur Verfügung, wenn im Dialog die Option Manuelle Flächenauswahl oder Manuelle Kantenauswahl aktiviert wurde.

Die Schaltfläche wird verwendet, wenn eine größere Anzahl miteinander verbundener Flächen / Flächenkanten ausgewählt werden sollen.

Anstatt jede einzelne Fläche auszuwählen, wählt man die erste und die letzte Fläche / Kante und klickt anschließend auf die Schaltfläche Feature abschließen.

Anschließend wählt das System alle Flächen zwischen der ersten und der letzten Fläche / Kante automatisch aus.
