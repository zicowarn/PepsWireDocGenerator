Erzeuge Figuren + Bearbeitungen von Features

Klicken Sie mit der rechten Maustaste auf und wählen Sie im Kontextmenü Erzeuge Figuren + Bearbeitungen von Features.

Diese Funktion erzeugt Bearbeitungsfiguren (2D Bearbeitungsgeometrien) sowie Bearbeitungsoperationen und übergibt diese an die Operationsliste.

Hinweis: Sie können Sie auch einzelne Features auswählen.

 

 

Bevor die Bearbeitungsfiguren und Bearbeitungen erzeugt werden, zeigt das System normalerweise (ist abschaltbar) einen Konfigurations-Dialog.

In diesem Dialog können Sie einstellen, ob Schnittschemen zugewiesen werden sollen, was an die PEPS Operationsliste übergeben werden soll und wie die erzeugten Bearbeitungsoperationen dargestellt werden sollen.
