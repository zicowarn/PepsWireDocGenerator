Kopiere Feature - In gleiche / neue Operation

 

Mit dieser Funktion können Sie ein oder mehrere Bearbeitungsfeature(s) kopieren.

Sie können die Features  in die aktuelle Operation kopieren oder die Features in eine neue Operation kopieren.

Die Option wird beispielsweise verwendet, um ein Feature zweimal,  mit unterschiedlichen Schneidparametern zu bearbeiten.

 

Hinweis:

Gruppierte Features können zur Zeit NICHT kopiert werden.   

 

Kopiere Feature in gleiche Operation

Kopiert ein oder mehrere gewählte Bearbeitungsfeature(s) an das Ende der aktuellen Operation.

 

 

Kopiere Feature in neue Operation

Erzeugt eine neue Operation und kopiert ein oder mehrere gewählte Bearbeitungsfeature(s) in die neue Operation

 

 

Tipp:

Wählen Sie mehrere aufeinander folgende Bearbeitungsfeature aus, indem Sie während des Auswählens die Umschalt-Taste gedrückt halten.

Wählen Sie nicht aufeinander folgende Bearbeitungsfeature aus, indem Sie während des Auswählens die Strg-Taste gedrückt halten.
