Aktiviere Feature

Nimmt Features die mit der Option Deaktiviere Feature aus der Bearbeitungsfolge herausgenommen wurden, wieder in die Bearbeitungsfolge auf.

Die Kennzeichnung des Features (deaktiviertes Objekt) im WIRE-Expert FeatureManager wird zurückgenommen.

 

Tipp:

Wählen Sie mehrere aufeinander folgende Features aus, indem Sie während des Auswählens die Umschalt-Taste gedrückt halten.

Wählen Sie mehrere nicht aufeinander folgende Features aus, indem Sie während des Auswählens die Strg-Taste gedrückt halten.
