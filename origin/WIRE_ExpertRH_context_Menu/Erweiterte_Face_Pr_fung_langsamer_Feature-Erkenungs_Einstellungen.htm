Erweiterte Erkennung (langsamer)

 

Bei einer automatischen Feature-Erkennung kann es passieren, das OPTICAM Feature erkennt (z.B. Sackbohrungen auf der Bauteilunterseite), die eigentlich nicht erkannt werden sollten.

Das folgende Bild soll eine solche Situation zeigen. Links das Ergebnis OHNE die Option Erweiterte Erkennung (langsamer) und rechts mit.

Die fünf roten Pfeile kennzeichnen die unerwünschten Feature (Sackbohrungen auf der Bauteilunterseite).

Der blaue Kreis markiert die Ausschnittsvergrößerung im dritten Bild.

 

Die Option Erweiterte Erkennung (langsamer) kann in solchen Fällen helfen.

Die Erweiterte Erkennung (langsamer) führt bei einer automatischen Feature-Erkennung, zusätzliche interne Tests aus, die unerwünschte Features entfernen oder deren Anzahl reduziert.

 

Hinweis:

Das Anwenden der Erweiterten Erkennung  kann die Berechnungszeit der automatischen Feature-Erkennung deutlich erhöhen.
