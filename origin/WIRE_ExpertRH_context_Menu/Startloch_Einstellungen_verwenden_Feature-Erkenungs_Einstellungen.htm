Startloch Einstellungen verwenden

Ist diese Option aktiv, werden die im Dialog Startloch-Einstellungen festgelegten Regeln zur Positionierung des Startloches verwendet.

Ist die Option NICHT aktiv, wird das interne Regelwerk (nicht vom Anwender anpassbar) verwendet.

 

Das folgende Bild zeigt den Dialog Startloch-Einstellungen.

Klicken Sie auf die einzelnen Bereiche des folgenden Bildes, um Hilfetexte aufzurufen.
