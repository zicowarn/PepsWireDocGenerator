Lösche Operation

Löscht die aktuell im WIRE-Expert Feature Manager gewählte(n) Operation(en).

 

Tipp:

Sie können Operationen im Feature Manager auswählen, indem Sie auf ihre Namen klicken.

Wählen Sie mehrere aufeinander folgende Operationen aus, indem Sie während des Auswählens die Umschalt-Taste gedrückt halten.

Wählen Sie nicht aufeinander folgende Operationen aus, indem Sie während des Auswählens die Strg-Taste gedrückt halten.
