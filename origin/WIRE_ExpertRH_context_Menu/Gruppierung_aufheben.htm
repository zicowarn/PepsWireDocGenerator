Gruppierung aufheben

Im WIRE-Expert Feature Manager ist es möglich, mehrere Features zusammenzufassen zu einem gruppierten Feature.

Ist dies der Fall und soll ein gruppiertes Feature zurückverwandelt werden in einzelne Features, verwendet man die Funktion Gruppierung aufheben.

 

 

Das Bild oberhalb zeigt links ein gruppiertes Feature und rechts die durch Aufheben der Gruppierung entstandenen Features.

 

So wird es gemacht:

  • Klicken Sie im Feature Manager das gruppierte Feature an, dessen Gruppierung aufgehoben werden soll.

  • Klicken Sie die rechte Maustaste und wählen Sie im Menü die Option Gruppierung aufheben.

Das System löst das Feature auf in einzelne Features.
