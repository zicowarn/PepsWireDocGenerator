Gruppiere Features

Im WIRE-Expert Feature Manager ist es möglich, mehrere Features zusammenzufassen zu einem Feature.

Dies reduziert die Anzahl der Einträge im WIRE-Expert Feature Manager wenn viele identische Geometrien bearbeitet werden müssen.

 

 

Das Bild oberhalb zeigt links das Ergebnis ohne gruppieren und rechts mit gruppieren.

 

So wird es gemacht:

  • Klicken Sie im Feature Manager nacheinander alle Features an, die zusammengefasst werden sollen. Halten Sie während der Auswahl die Umschalt-Taste gedrückt.

  • Klicken Sie die rechte Maustaste und wählen Sie im Menü die Option Gruppiere Features.

Das System fasst die gewählten Features zusammen zu einem Feature.
