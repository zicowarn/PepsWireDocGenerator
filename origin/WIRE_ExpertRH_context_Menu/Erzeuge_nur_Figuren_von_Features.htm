Erzeuge nur Figuren von Features

Diese Funktion wandelt die gewählten Features um und erzeugt ausschließlich Bearbeitungsfiguren (2D Bearbeitungsgeometrien).

 

Im Gegensatz zur Funktion Erzeuge Figuren + Bearbeitungen von Features erzeugt diese keine Bearbeitungsoperationen.

Die fehlenden Bearbeitungsoperationen müssen manuell programmiert werden.
