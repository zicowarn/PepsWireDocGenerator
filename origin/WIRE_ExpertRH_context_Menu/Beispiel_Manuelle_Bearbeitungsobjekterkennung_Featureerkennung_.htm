Manuelle Feature-Erkennung

Dieses Beispiel soll zeigen, wie man die Schneidbearbeitungen halbautomatisch erzeugt, indem man die zu bearbeitenden Flächen manuell, mit Hilfe der Maus auswählt.

 

Klicken Sie mit der rechten Maustaste im Feature Manager auf den Zweig .

Es klappt ein Kontextmenü auf. Wählen Sie den Befehl Manuelle Feature-Erstellung.

 

 

Es wird der Dialog Neues Feature angezeigt. Aktivieren Sie im Dialog die Option Manuell mit automatischer Flächenverbindung.

 

Bei Wahl dieser Option klickt man eine Fläche an und das System findet automatisch alle erodierbaren Flächen, die mit der gewählten Fläche verbunden sind.

 

 

Klicken Sie eine Fläche des inneren Durchbruchs an.

 

 

Das System erkennt automatisch alle Flächen die zum gewählten Durchbruch gehören.

 

 

Klicken Sie eine Fläche der äußeren Geometrie an.

Das System erkennt wieder automatisch alle Flächen die zur Außenkontur gehören.

Übernehmen Sie die Auswahl mit .

 

Für jedes gewählte Objekt erzeugt das System ein separates Feature im FeatureManager. Da zwei Objekte gewählt wurden, müssten 2 Features existieren:

 

 

Jetzt noch aus den Features Bearbeitungsoperationen und Figuren erzeugen.

 

Klicken Sie mit der rechten Maustaste im Feature Manager auf den Zweig und wählen Sie im Kontextmenü Erzeuge Figuren + Bearbeitungen von Features.

Diese Funktion erzeugt Bearbeitungsfiguren (2D Bearbeitungsgeometrien) sowie Bearbeitungen und übergibt diese an die Operationsliste.

Hinweis: Sie können Sie auch einzelne Features auswählen.
