Lösche Feature

Löscht die aktuell im WIRE-Expert FeatureManager gewählten Features.

 

Tipp:

Sie können Feature im FeatureManager auswählen, indem Sie auf deren Namen klicken.

Wählen Sie mehrere aufeinander folgende Features aus, indem Sie während des Auswählens die Umschalt-Taste gedrückt halten.

Wählen Sie nicht aufeinander folgende Features aus, indem Sie während des Auswählens die Strg-Taste gedrückt halten.
