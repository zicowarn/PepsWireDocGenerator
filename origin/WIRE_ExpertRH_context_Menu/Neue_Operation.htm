Neue Operation

Diese Funktion wird verwendet, um einen Eintrag für eine neue Operation im WIRE-Expert Feature Manager zu erzeugen.

Jede neue Operation erzeugt eine separate NC-Ausgabedatei und enthält Features.

Durch das Erstellen von mehreren Operationen können Sie innerhalb eines WIRE-Expert Programms mehrere separate NC-Ausgabedateien erzeugen.

 

 

Das Bild oberhalb zeigt zwei Operationen.
