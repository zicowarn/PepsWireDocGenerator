Boxauswahl verwenden

 

Ist diese Option aktiviert, können Sie VOR dem Start der Automatischen Featureerkennung eine Auswahlfenster "aufziehen".

Mit Hilfe des Auswahlfenster können Sie einen Teilbereich im 3D Modell auswählen, in dem die  Automatischen Featureerkennung ausgeführt werden soll.

 

Die folgenden Bilder zeigt links wie ein rotes Auswahlfenster, mit dessen Hilfe zwei Bohrungen ausgewählt werden sollen.

Das rechte Bild zeigt das Ergebnis der  Automatischen Featureerkennung, nämlich zwei bearbeite Bohrungen.
