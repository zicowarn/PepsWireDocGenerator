Strategien

Hier stellen Sie ein, welche Objekte die Feature-Erkennung erkennen soll.

 

Neue Operation für jeden Body erzeugen

Ist diese Option aktiv und enthält die aktuelle Datei mehrere Volumenkörper, erzeugt WIRE-Expert für jeden existierenden Volumenkörper eine separate Operation im FeatureManager.
Beispiel:

Existieren in einer Datei zwei Volumenkörper, so kann durch aktivieren der Option Neue Operation für jeden Body erzeugen, für jeden Volumenkörper eine separate NC-Ausgabedatei erzeugt werden, da für jeden Körper eine eigene Operation im FeatureManager erzeugt wird.

 

 

Innere & Äussere Features

Die automatische Feature-Erkennung erkennt alle erodierbaren Geometrien innerhalb des Volumenkörpers plus die Außengeometrie des Körpers.

 

Nur Innere Features (Matritze)

Die automatische Feature-Erkennung erkennt ausschließlich alle erodierbaren Geometrien innerhalb des Volumenkörpers. Die Außengeometrie des Körpers bleibt unbearbeitet.

 

Nur Äußere Features (Stempel)

Die automatische Feature-Erkennung erkennt ausschließlich die Außengeometrie des Volumenkörpers. Dies ist die bevorzugte Einstellung für die Bearbeitung von Schneidstempeln.
