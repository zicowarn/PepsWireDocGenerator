Automatische Feature-Erkennung

Dieses Beispiel soll zeigen, wie man die Schneidbearbeitungen automatisch erzeugt, mit Hilfe der automatischen Feature-Erkennung.

 

Klicken Sie mit der rechten Maustaste im Feature Manager auf den Zweig Operation1.

 

Es klappt ein Kontextmenü auf. Wählen Sie den Befehl Automatische Feature-Erkennung.

 

 

Es wird der Dialog Feature-Erkennung angezeigt. In dem Dialog stellen Sie welche Objekte die automatische Erkennungsroutine erkennen soll.

Im folgenden Beispiel würde die Außenkontur des Modells nicht bearbeitet (Nur innere Feature (Matrize)).

 

Um die Feature-Erkennung einzustellen, klicken Sie auf die Schaltfläche Feature-Erkennung Einstellungen.

Dies öffnet einen Dialog, im Sie die Feature-Erkennung einstellen können.

 

 

Stellen Sie sicher, dass identische Feature zusammengefasst werden zu einem Feature und aktivieren Sie Identische Feature gruppieren.

Das System durchsucht den Volumenkörper nach allen erodierbaren Geometrien (Feature-Erkennung) und erzeugt einen Bearbeitungsvorschlag mit den im Vorfeld eingestellten Schneidparametern.

 

Das System durchsucht den Volumenkörper nach allen erodierbaren Geometrien (Featureerkennung).

 

 

Für jede erodierbare Geometrie erzeugt das System ein separates Feature im FeatureManager:

 

 

 

Jetzt noch aus den Features Bearbeitungsoperationen und Figuren erzeugen.

 

Klicken Sie mit der rechten Maustaste im Feature Manager auf den Zweig und wählen Sie im Kontextmenü Erzeuge Figuren + Bearbeitungen von Feature.

Diese Funktion erzeugt Bearbeitungsfiguren (2D Bearbeitungsgeometrien) sowie Bearbeitungen und übergibt diese an die Operationsliste.

Hinweis: Sie können Sie auch einzelne Feature auswählen.
