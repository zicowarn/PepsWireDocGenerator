Manuelle Achsensteuerung

 

Mit Hilfe der Manuellen Achsensteuerung können Sie die Linear- und Rotationsachsen der Maschine, manuell durch Tastatureingaben oder mit Schiebereglern bewegen und kontrollieren.

 

 

Hauptvorteile:

Es ist überprüfbar, ob die Achsengrenzen korrekt eingestellt sind.

Es ist überprüfbar, ob die Kollisionsprüfung korrekt eingestellt ist.

 

Beschreibung:

Ermittelt die Achsengrenzen.

Ermittelt die aktuelle Achsenposition.

Verändern der Achsenwerte, entweder über Eingabefelder, verschieben der Schieberegler oder durch einen "Rechtsklick" auf einen der Schieberegler, um die Eingaben in einem Dialog zu tätigen (Siehe Beschreibung unterhalb).

Die Eingabefelder unterhalb der Schieberegler, zeigen den aktuellen Wert einer Achse und werden bei einer Simulation kontinuierlich aktualisiert.

Um von einem Feld von links nach rechts in das nächste zu springen, verwendet man die TAB-Taste. Um von einem Feld von rechts nach links in das nächste zu springen, verwendet man die SHIFT+TAB-Taste.  

If you are jumping on a field that is not visible, the Axis Control tab is automatically scrolling to show the respective field.

XXX Wenn Sie auf ein unsichtbares Feld springen, XXX

 

XXX Numeric stepper actions:

The values can be copied pressing CTRL+C and paste command can be applied using CTRL+V (Windows standard keys)

Die Werte lassen sich kopieren durch Drücken der Tastenkombination CTRL+C und wieder einfügen mit Hilfe der Tasten CTRL+V (MS-Windows Standardfunktionen)

If upper and lower arrows from numeric steppers are used, these will increment with 0.1 for metric simulations and with 0.01 for inch simulations

XXX

If you want to apply Cut command (CTRL+X) in numeric stepper, after the value will disappear from the used field, 0 (zero) will be put instead.

XXX

 

Kontextmenü auf rechter Maustaste

Klicken Sie mit der rechten Maustaste auf einen der Achsen-Schieber, um den folgenden Dialog zu öffnen. Der Dialog erlaubt eine bestimmte Achsenposition einzugeben oder die Achsenposition auf 0 zu setzen.

 

Temporary Relative Jog Values: Mit dieser Option können Sie alle Achsenwerte auf 0 setzen ohne das Maschinenmodell zu bewegen.

Auf diese Weise können Sie das Maschinenmodell solange temporär auf einen bestimmten Wert bewegen, wie die Option aktiviert ist. Schließt man den Dialog wird der Effekt der Option beendet.   

Alle auf 0: Mit dieser Option können Sie alle Achsen auf 0 zurücksetzen

Auf Anfangswerte zurücksetzen: Mit dieser Option können Sie alle Achsen auf deren ursprünglich eingestellten Werte zurücksetzen. Dies sind die der Maschinendefinitionsdatei gespeicherten Werte.

Alles in Zwischenablage kopieren: Mit dieser Option können Sie die Einstellwerte aller Achsen kopieren.
