Aktuellen Simulationstand abspeichern

Normalerweise startet die Simulation immer am Programmanfang. Bei umfangreichen Programmen möchte mancher Anwender die Simulation jedoch an einer bestimmten Position beenden.

Beim nächsten Aufruf der Simulation soll die Simulation an dieser Stelle starten und nicht am Programmanfang.

 

So wird es gemacht:

Aktuellen Simulationstand abspeichern

  • Simulieren Sie das Programm bis ca. zur Mitte des Programms.

  • Klicken Sie im Dialogbereich Speichern auf die Schaltfläche Simulation.

  • Beenden Sie die Simulation mit der Schaltfläche Ende.

  • Starten Sie die Simulation erneut. Es erscheint eine Meldung.

  • Bestätigen Sie mit Ja.

  • Die Simulation wird an die Stelle springen, an welcher der Simulationstand gespeichert wurde.

  • Starten Sie die Simulation. Sie werden feststellen, dass die Simulation an der Stelle startet, an welcher der Simulationstand gespeichert wurde.
