Bild von fertiger Simulation speichern

Diese Option erzeugt am Ende einer Simulation eine Jpeg-Grafikdatei der aktuellen Simulationsansicht.

Die Datei wird im Verzeichnis der aktuellen *.vdm Datei gespeichert.

Der Name der Datei ist identisch mit der aktuellen *.vdm-Datei, wird aber erweitert um das Suffix ".sim". Beispiel: Grundplatte_sim.jpeg.
