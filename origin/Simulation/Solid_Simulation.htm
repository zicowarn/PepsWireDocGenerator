Solid Simulation

 

Programmsimulation in Form einer Solid Simulation mit Simulation des Materialabtrags.

In dieser Simulationsform zeigt das System den Schneiddraht, die obere, untere Düse, das Werkstück, das Rohteil und den Materialabtrag als Solidmodell.

 

 

Die Simulation wird über 3 beliebig verschiebbare Dialoge gesteuert. Dies sind der Steuerdialog, der Listendialog und der Ansichtdialog.

 

Klicken Sie auf die einzelnen Bereiche des folgenden Bildes, um Hilfetexte aufzurufen

 

So wird es gemacht:

Kontrolle der Schneidtechnologie
Während der Simulation zeigt das System zusätzlich den Dialog Registerkontrolle. Der Dialog zeigt die Technologieeinstellungen des momentan simulierten Schnitts.

Ausfallteilüberprüfung
Bei der Simulation einer Drahterodierbearbeitung kann es zu Situationen kommen, in der Ausfallteile (Abfall löst sich vom Werkstück) erzeugt werden.

PEPS kann das Ausfallteil grafisch kennzeichnen und testen, ob sich das Ausfallteil entfernen lässt. Weitere Einzelheiten finden Sie siehe: Ausfallteilüberprüfung.

Ändern der Simulationsansicht
Während einer Solid-Simulation können Sie die gezeigte Ansicht beliebig mit Hilfe der Maus verändern. Die Ansicht lässt sich Rotieren, Zoomen und Verschieben.

Im Kapitel Interaktives Zoomen und Rotieren im PEPS Grafikfenster finden Sie Einzelheiten über die Vorgehensweise.

Bei geometrisch komplexen Werkstücken können Sie das Rotieren der Ansicht beschleunigen, indem Sie während des Rotierens die Taste Strg gedrückt halten.
