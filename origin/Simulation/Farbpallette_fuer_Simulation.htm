Farbpalette für Simulation

 

Diese Schaltfläche führt zu Aufruf des Standard Windows Farbpaletten Dialogs.

In dem Dialog können Sie eine von 48 Farben auswählen und für die Simulation verwenden.

Die Schaltfläche Farben definieren >> erlaubt das Mischen eigener Farben.

 

PEPS-Hintergrund verwenden

Verwendet als Hintergrundfarbe für das Simulationsfenster die gleiche Farbe wie für das PEPS-Grafikfenster.
