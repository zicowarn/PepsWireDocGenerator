Sichtbare Facettierung

Diese Einstellung findet Anwendung, um die Simulationsgeschwindigkeit von komplexen Werkzeugen und Werkzeughaltern, die mit einem sehr kleinem Facettierungswert simuliert werden, zu verbessern.

Durch Erhöhen des Wertes für die Sichtbare Facettierung, zeigt das System Werkzeuge/Halter mit weniger Facetten an, ohne das dies einen Einfluss auf einen Zielteilvergleich hätte.

Stellt man den Wert für die Sichtbare Facettierung auf 0 ein, verwendet das System die Facettierungseinstellungen für das Werkzeug und den Werkzeughalter.
