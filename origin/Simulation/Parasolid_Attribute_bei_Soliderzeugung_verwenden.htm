Parasolid-Attribute bei Soliderzeugung verwenden

Startet man die Simulation konvertiert das System alle Parasolid-Elemente (Solid-Bodies der Rohteile, Zielteile, Spannmittel, etc...) die in der Simulation verwendet werden.

Das Aktivieren dieser Option führt dazu, dass die Eigenschaften der Parasolid-Elemente (Namen, Farben etc. der Solid-Bodies) von der Simulation verwendet werden.
