Ausfallteil prüfen

In diesem Dialogbereich stellen Sie die Parameter ein, die das System bei einer Ausfallteilüberprüfung verwendet (System prüft, ob sich das beim Erodieren entstehende Ausfallteil entnehmen lässt).

Die Ausfallteilüberprüfung wird mit Hilfe des Dialogs Ausfallteilüberprüfung gesteuert.

Die folgenden Parameter lassen sich einstellen:

Schrittweite für Kollisionsprüfung

Bestimmt den inkrementalen Abstand zwischen den einzelnen Kollisionsprüfungen, die das System bei einer Ausfallteilüberprüfung durchführt.

Beispiel: Wird ein Ausfallteil aus einem Werkstück von 30 mm Höhe entfernt und ist eine Schrittweite von 10 mm eingestellt, finden 3 Kollisionsprüfungen statt.

Anzahl der Kollisionsprüfungen zwischen jeder Anzeige

Diese Einstellung bestimmt, wie viele Ebenen die Ausfallteilüberprüfung graphisch anzeigt.

Ist die Schrittweite für Kollisionsprüfung so eingestellt, dass das System die Ausfallteilüberprüfung auf 6 Ebenen durchführt und ist der Parameter Anzahl der Kollisionsprüfungen zwischen jeder Anzeige auf 2 eingestellt, dann zeigt das System die Ausfallteilüberprüfung auf 3 Ebenen graphisch an. Stellt man die Anzahl der Kollisionsprüfungen auf 1 ein, würde die Ausfallteilüberprüfung an 6 Ebenen angezeigt.

Verzögerung nach jedem Schnitt (msek.)

Stellt ein, wie lange jede einzelne Ausfallteilüberprüfung graphisch angezeigt wird. Mit Hilfe dieser Einstellung können Sie die Simulation der Ausfallteilüberprüfung verlangsamen/beschleunigen.
