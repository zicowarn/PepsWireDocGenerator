Bearbeitungsinformationen auf Faces speichern

Mit dieser Option können Sie die Möglichkeit, Bearbeitungsinformationen auf den geschnittenen Flächen zu speichern (z.B. welche Operation hat eine Kavität geschnitten), ein- und ausschalten.

Nutzen Sie die Funktion Informationen aus Simulationsgrafik abfragen, um die Informationen abzufragen.

Ist die Option nicht aktiv, läuft die Simulation schneller, da keine Informationen auf den bearbeiteten Flächen gespeichert werden.

 

Hinweis:

Änderungen an dieser Optionseinstellung werden erst bei der nächsten Simulation wirksam.
