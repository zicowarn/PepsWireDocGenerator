Farben Register (Einstellungen)

Das Register Farben ruft einen Dialog auf, in der Sie die Farben für die Solid-Simulation einstellen können.

Sie können jedes der Farbfelder individuell auswählen und einstellen, indem Sie verschiedene Anteile von Rot, Grün oder Blau zuweisen.

Durch Anklicken der Schaltfläche Farbpalette ruft das System einen Dialog auf, in der Sie direkt eine von 48 verschiedenen Farben auswählen können.

Der folgende Dialog zeigt die Option WKZ.-Eilgänge aktiv. Als Farbe wurde hauptsächlich rot (255 rote Farbanteile) zugewiesen.

 

Klicken Sie auf die einzelnen Bereiche des folgenden Bildes, um Hilfetexte aufzurufen.
