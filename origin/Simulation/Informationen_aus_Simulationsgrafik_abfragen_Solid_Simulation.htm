Informationen aus Simulationsgrafik abfragen

Mit Hilfe dieser Ikonen können Sie beliebige Flächen in der Simulationsgrafik anklicken, wobei das System Informationen über das gewählte Element liefert.

: Gibt allgemeine Informationen über die Art (z.B. Rohteil, Düse) des gewählten Objektes aus.
: Erlaubt das Anklicken einzelner Flächen und gibt erweiterte Informationen (bearbeitet / unbearbeitet / welche Operation hat die Fläche bearbeitet) über die gewählte Fläche aus.
: Erlaubt das Bemaßen von 2 Punktpositionen in der Simulationsgrafik.
