Standard Simulation

 

Programmsimulation OHNE Simulation des Materialabtrags. In dieser Simulationsform zeigt das System den Schneiddraht, die obere, untere Düse und das Rohteil als Solidmodell.

Im Unterschied zur Solid-Simulation wird der Materialabtrag NICHT dargestellt und es ist KEIN Zielteilvergleich und KEINE Kollisionskontrolle möglich.

 

 

Die Simulation wird über einen beliebig verschiebbaren Dialog gesteuert.

 

Klicken Sie auf die einzelnen Bereiche des folgenden Bildes, um Hilfetexte aufzurufen

 

 

So wird es gemacht:

Kontrolle der Schneidtechnologie
Während der Simulation zeigt das System zusätzlich den Dialog Registerkontrolle. Der Dialog zeigt die Technologieeinstellungen des momentan simulierten Schnitts.

Ändern der Simulationsansicht
Während einer Solid-Simulation können Sie die gezeigte Ansicht beliebig mit Hilfe der Maus verändern.

Die Ansicht lässt sich Rotieren, Zoomen und Verschieben. Im Kapitel Interaktives Zoomen und Rotieren im PEPS Grafikfenster finden Sie Einzelheiten über die Vorgehensweise.

Bei geometrisch komplexen Werkstücken können Sie das Rotieren der Ansicht beschleunigen, indem Sie während des Rotierens die Taste Strg gedrückt halten.
