Schnellste Methode für Bildneuaufbau verwenden

Ist diese Option aktiv, baut das System bei einem Neuaufbau der Simulationsgrafik, nur den sichtbaren Bereich des Grafikfenster neu auf.

Beispiel:

Ein Dialog überdeckt die Simulationsgrafik. In diesem Fall wird der Bereich hinter dem Dialog nicht neu aufgebaut.
