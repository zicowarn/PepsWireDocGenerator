Werkzeugwege anzeigen

Ein- oder ausschalten der Werkzeugweggrafik (Verfahrwege und Positionierbewegungen). Die Checkbox erlaubt die Einstellung von 3 Simulationsmodi.

Jeder Mausklick auf die Checkbox aktiviert einen der 3 Modi:

 

1. Checkbox aktiviert: Es werden alle Werkzeugwege angezeigt.

2. Checkbox aktiviert (aber ausgegraut): Es wird nur der aktuelle Werkzeugweg angezeigt.

3. Checkbox deaktiviert: Es werden keine Werkzeugwege angezeigt.
