Stop- / Düsensymbole

Stop-Symbole: Ist diese Option aktiv, zeigt das System bei einer Simulation alle Stops (M00, M01) in der Form von Symbolen an.

 

 

Düsen-Symbole: Ist diese Option aktiv, zeigt das System bei einer Simulation die obere und untere Drahtführung symbolisch an, wenn der Draht eingefädelt ist.
