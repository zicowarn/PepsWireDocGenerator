Normale der Figur / In Z

Legt die Projektionsart fest. Ist das Register Normale zur Figur aktiviert, erfolgt die Projektion im rechten Winkel zur der Figur auf die XY-Ebene.

Ist das Register In Z aktiviert, erfolgt die Projektion lotrecht von der Figur auf die XY-Ebene. (Siehe Zeichnung Option Maximale Abweichung ).

Die folgende Zeichnung zeigt die zwei möglichen Projektionstypen. Die obere zeigt eine Projektion bei aktiviertem Register Normale der Figur.

Hier erfolgt die Projektion im rechten Winkel von der Figur auf die XY-Ebene.

Die Zeichnung darunter zeigt das Ergebnis, wenn das Register In Z aktiviert ist. Dann erfolgt die Projektion lotrecht von der Figur auf die XY-Ebene.
