Geometrie Parameter

Diese Option ruft einen Dialog auf, mit dem Sie die Geometrie-Eigenschaften von Flächen, Kanten und Vertexe überprüfen können.

Auf diese Weise kann man beispielsweise den Start-\Endpunkt oder den Winkel einer Modellkante überprüfen.

Zudem wird analysiert, ob eine Kante oder Fläche erodierbar ist oder nicht.
