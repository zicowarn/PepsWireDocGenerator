Eine Bearbeitungsfigur auf eine Punktegruppe (G1 ...) verteilen

  • Zeichnen Sie die Bearbeitungsfigur

  • Bestimmen Sie die Positionen auf welcher die Bearbeitungsfigur kopiert werden soll, mit Hilfe des Registers Gruppe der Dialogbox NURBS-Kurven und Surfaces.

  • Der Befehl erzeugt eine Punktegruppe (G1 ..).

  • Rufen Sie im Menü Vorbereitung \ Diverses den Befehl Kreise/Bögen ==> Mehrfachfigur auf.

  • Klicken Sie zuerst die Punktegruppe (G1 ..) an und anschließend die zu kopierende Bearbeitungsfigur.

  • Das System nimmt die gewählte Figur und kopiert diese auf die vorhandenen Punktegruppe (G1 ..).

  • Beachten Sie, dass die Start(loch)position der Figur deckungsgleich auf den Gruppenpunkten liegt.
