Bogenradius

Soll das System Bögen die kleiner sind, als ein bestimmbarer Radius, bei der Figurerzeugung farblich kennzeichnen, dann können Sie dies mit den Optionen Zeige Radien kleiner als und Graphisch in ... einstellen.

Wenn Sie einen Report über kleine Radien ausgegeben wollen, müssen Sie die Option In Report aktivieren.

 

 

Mit den Optionen im Feld Zeige Radien größer als, können Sie Bögen die größer sind, als ein bestimmbarer Radius farblich kennzeichnen und in den Report einbeziehen.
