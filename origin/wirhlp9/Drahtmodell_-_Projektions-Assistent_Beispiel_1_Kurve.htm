Projektion einer Kurve unter einem konstanten Winkel

Das folgende Beispiel zeigt, wie man von einer 3D Kurve zwei XY-planare Regelflächengeometrien ableitet.

Dies ermöglicht, die 3D Geometrie unter einer konstanten Konik, als Regelfläche zu schneiden.

Verwenden Sie die Schaltfläche Wähle Kurve >> und selektieren Sie die Geometrien der 3D Kurve.

Hinweis: Erlaubte Geometrietypen sind: Linien/Bögen, Kanten und NURBS-Kurven.

Das Rote Ausrufezeichen verändert sich zu einem Grünen Haken der anzeigt, dass eine passende 3D Kurve gewählt wurde.

 

Wählen Sie die Schaltfläche Weiter >, zum Aufruf des nächsten Dialogs:

 

Geben Sie im Feld Winkel den benötigten Konikwinkel ein

Die beiden bebilderten Schaltflächen dienen zur Auswahl der Konikrichtung.

Wählen Sie die Schaltfläche Weiter >>, zum Aufruf des nächsten Dialogs:

 

 

Obere Projektion

Geben Sie in diesem Feld die Z-Höhe ein, auf welche die oberen Regelflächengeometrie projiziert werden soll.

UntereProjektion

Geben Sie in diesem Feld die Z-Höhe ein, auf welche die untere Regelflächengeometrie projiziert werden soll.

Bereiche mit gleicher Elementanzahl angleichen

Ist die Option aktiv, wird eine proportionale Synchronisation für die unter/obere Regelflächengeometrie ausgeführt.

Falls nicht, werden die Elementenden der unter/obere Regelflächengeometrie miteinander verbunden.

Figuren erzeugen

Aktivieren Sie diese Option wenn Figuren anstatt Linien/Bögen erzeugt werden sollen.

Vorgabe für Figurnamen

Falls Figuren erzeugt werden sollen, hier den Namen der Figur (z.B. Fig_) eingeben. Das System macht daraus Fig_001 und Fig_002.

Projektionstoleranz

Steuert wie genau die Linien/Bögen der projizierten Geometrie werden.

Je kleiner der Wert, desto genau die projizierten Geometrien. Wir empfehlen einen Wert von 0.001.

 

Klicken Sie nun auf die Schaltfläche Projezieren, um die beiden planaren Geometrien abzuleiten.

Das folgende Bild zeigt in der Mitte die zu projizierende Geometrie und darüber/darunter die beiden projizierten Regelflächengeometrien.  

 

 

Beenden Sie den Dialog mit Fertig.

 

Die beiden projizierten Regelflächengeometrien sollten jetzt noch in Bearbeitungsfiguren inkl. Startpunkten umgewandelt werden.

Anschließend können Sie die Bearbeitungsfiguren für die Regelflächenprogrammierung verwenden.
