Startloch (Linien/Bögen => Bearbeitungsfigur)

In diesem Feld wird die Startposition einer Figurbearbeitung festgelegt. Dies kann das Zentrum einer Startlochbohrung sein oder eine andere Position, weit genug von der Bearbeitungsfigur entfernt.

Von dieser Position aus beginnt die Bearbeitung einer Figur mit einer Vorschubbewegung.

Die Startposition wird normalerweise direkt als XY-Koordinate eingegeben oder mit der Schaltfläche und mit Hilfe der Fangfunktionen Endpunkt - Koordinaten in der Zeichnung ermittelt.
