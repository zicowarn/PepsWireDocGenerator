Report anzeigen

Legt fest, ob das System nach der Figurerzeugung einen Report über Umfang, Fläche, Schwerpunkt, kleinster Radius etc. der Figur ausgeben soll.

Der Report wird in folgenden Dialog angezeigt.

 

 

Welche Informationen der Report ausgibt, können Sie in den Feldern Bogenradius, Elementlänge und Zusätzliche Report Optionen einstellen.
