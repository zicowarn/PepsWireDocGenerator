NURB-Surface/Kurve projizieren

Mit dem Register NURBS-Kurve dieses Befehls können Sie eine 3D-NURBS-Kurve so projizieren, dass zwei XY-planare Figuren entstehen, die auf einer bestimmbaren Referenzhöhe bzw. Sekundärebene liegen.

Mit dem Register NURBS-Surface können Sie ein NURBS-Surface, dessen äußere Berandungskurve nicht auf der Referenzhöhe bzw. Sekundärebene liegt, so projizieren, dass zwei neue Figuren entstehen, die auf einer bestimmbaren Referenzhöhe bzw. Sekundärebene liegen.
