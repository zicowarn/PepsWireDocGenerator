In ISO Linie/Bogen/Linie umwandeln

Es gibt Drahterosionsmaschinen, bei denen es nicht möglich ist, die Kombination Bogen/Bogen mit einer variablen Konik zu versehen.

Dieser Befehl erlaubt es, in einem Bogen/Bogen Übergang ein kleines Linienelement einzufügen, so dass die Kombination Bogen/Linie/Bogen entsteht.

 

 

Durch diesen Trick kann die Geometrie doch als variable Konik geschnitten werden.

 

So wird es gemacht:

  • Zeichnen Sie die Geometrie als Linien/Bogenelemente. Definieren Sie KEINE Bearbeitungsfigur.

  • Rufen Sie im Menü Vorbereitung \ Diverses den Befehl In ISO Linie/Bogen/Linie umwandeln auf.

  • Aktivieren Sie die Option Radien modifizieren / Tangentialität erhalten oder deaktivieren Sie die Option.

  • Bestimmen Sie im Feld Länge der eingefügten Linie die Länge der einzufügenden Linie.

  • Klicken Sie auf die Schaltfläche >> und wählen Sie die 2 Bogenelemente aus, zwischen denen die Linie eingefügt werden soll.

  • Beenden Sie den Auswahlmodus mit der Option Ende (siehe rechte Maustaste od. horizontale Ikonenleiste).

  • Verlassen Sie den Dialog und erzeugen Sie die Bearbeitungsfigur.
