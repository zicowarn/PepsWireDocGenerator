Report Register

In diesem Dialogregister können Sie einstellen, ob nach einer Figurerzeugung ein Report über Umfang, Fläche, Schwerpunkt, kleinster Radius etc. der Figur ausgeben werden soll.

Zusätzlich können Sie Geometrieelemente farblich kennzeichnen die kleiner oder größer sind als ein bestimmter Wert.

 

Klicken Sie auf die einzelnen Bereiche des folgenden Bildes, um Hilfetexte aufzurufen.
