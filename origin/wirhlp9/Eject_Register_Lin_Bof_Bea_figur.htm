Eject Register

Einige Drahterosionsmaschinen verfügen über eine Einrichtung zum automatischen Entfernen von Ausfallteilen (Ejector od. Ausfallteilentsorger).

Die Maschine positioniert dabei den Ausfallteilentsorger auf eine bestimmbare XY-Koordinaten über dem Ausfallteil, wobei der Entsorger einen Schlag auf das Ausfallteil ausführt.

Mit den Optionen in diesem Dialog können Sie bei der Figurerzeugung festlegen, an welcher XY-Koordinate der Ausfallteilentsorger positioniert werden soll.
