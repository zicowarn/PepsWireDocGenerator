Vorgabe für Figurnamen

Mit Hilfe dieses Feldes können Sie die Systemvoreinstellung für die Namen der erzeugten Bearbeitungsfiguren (ExpShape) zu verändern.

Verwenden Sie die Schaltfläche Eingaben speichern, um eine neue Vorgabe für den Figurnamen zu speichern.

Erzeugt das System mehrere Bearbeitungsfiguren, werden die Figurnamen automatisch nummeriert (ExpShape _001, ExpShape_002, ExpShape_003 ...).

 

Benutze Konturform als Namen

Ist diese Option aktiv, und erzeugt das System aus einer Standardgeometrie (z.B. Rechteck, Kreis, Langloch ...) eine Bearbeitungsfigur.

Dabei wird die interne Kennung der Standardgeometrie (z.B. Rechteck, Kreis, Langloch ...) als Figurname verwendet.

Zudem hängt das System an den Figurnamen die Maße der Standardgeometrie an.  

Beispiel:

Aus einem Rechteck 20 X 10 mm würde sich eine Bearbeitungsfigur mit dem Namen "Rechteck_20.0x10.0" ergeben.
