Startloch = Einfügeposition der Figur

Die Einfügeposition hat nur dann eine Bedeutung, wenn weitere Kopien einer vorhandenen Bearbeitungsfigur erzeugt werden sollen.

Dies kann mit der Optionen Kopieren oder Figur einfügen der Dialog Figur Optionen oder dem Befehl Kreise/Bögen => Mehrfachfigur im Menü Vorbereitung \ Diverses geschehen.

Ist die Option Startloch = Einfügeposition der Figur aktiv, verwendet das System bei Kopiervorgängen die Startlochposition als Einfügeposition.
