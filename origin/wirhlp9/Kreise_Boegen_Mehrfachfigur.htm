Kreise/Bögen ==> Mehrfachfigur

Dieser Befehl hat zwei Aufgaben:

1.) Bohrungen gleichen Durchmessers suchen und Bearbeitungsfiguren erzeugen.

In einigen Fällen kann die Situation auftreten, dass eine größere Anzahl von Bohrungen mit gleichen Durchmessern geschnitten werden müssen.

Normalerweise ist es dabei notwendig, aus jeder Bohrung eine separate Bearbeitungsfigur zu erzeugen. Dies kann bei einer größeren Anzahl gleicher Bohrungen recht zeitaufwendig sein.

 

 

Dieser Befehl erlaubt es, eine beliebige Anzahl Bohrungen gleichen Durchmessers mit einem Befehlsaufruf, automatisch in Bearbeitungsfiguren umzuwandeln.

2.) Eine Bearbeitungsfigur auf eine Punktegruppe (G1 ...) verteilen.

Manchmal soll eine vorhandene Bearbeitungsfigur auf einer Anzahl von Punktposition vervielfältigt werden.

Normalerweise ist es dabei notwendig, an jeder Position eine separate Bearbeitungsfigur zu erzeugen.

 

 

Mit dem Befehl ist es möglich, eine vorhandene Bearbeitungsfigur über eine Punktegruppe zu verteilen (Siehe Register Gruppe im Dialog CAD \ 3D-Geometrie erzeugen \ NURBS-Kurven und Surfaces).
