Schnittschemen

Dieser Dialogbereich legt fest, ob und wann Schnittschemen (abgespeicherte Bearbeitungsstrategien z.B. Technologiewerte, Anzahl der Schnitte, Breite der Anbindung, Bearbeitungsstrategie etc.) zugewiesen werden sollen.

Keine Abfrage:

Es wird kein Dialog zur Auswahl eines Schnittschemas angezeigt.


Schnittschemen Abfrage

Einmal beim Start

Es wird nur einmal der Dialog zur Auswahl eines Schnittschemas angezeigt.

Für jede Operation:

Werden vom WIRE-Expert mehrere Operationsstarts in der PEPS Operationsliste erzeugt, wird für jeden neuen Operationsstart ein Dialog zur Auswahl eines Schnittschemas angezeigt.


Für jedes Feature

Für jedes erzeugte Feature (erkannte Bearbeitungsgeometrie) wird ein Dialog zur Auswahl eines Schnittschemas angezeigt.
