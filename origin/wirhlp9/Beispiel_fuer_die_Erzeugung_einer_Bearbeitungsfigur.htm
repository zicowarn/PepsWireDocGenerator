Beispiel für die Erzeugung einer Bearbeitungsfigur

Fast alle Bearbeitungsoperationen der Module Erodieren / 2,5D Fräsen / Drehen basieren auf dem Geometrietyp Bearbeitungsfigur.

Sie sollten unbedingt mit der Erzeugung von Konstruktionen auf der Basis von C-Linien, C-Kreisen und Linien-/Bogenelementen vertraut sein, bevor Sie weiterlesen.

Um eine Bearbeitung, ausführen zu können, müssen Sie die vorhandenen Linien-/Bogenelemente zu einer Bearbeitungsfigur zusammenfassen. Dies geht wie folgt:

  • Es soll aus der folgenden Geometrie (müssen Linien-/Bogenelemente sein) eine Bearbeitungsfigur erzeugt werden.

 

  • Wählen Sie im Menü Vorbereitung den Befehl Linien/Bögen => Bearbeitungsfigur.

  • Geben Sie im Feld Startloch die Position ein, an welcher der Draht einfädeln soll oder ermitteln Sie die Koordinaten in der Zeichnung mit Hilfe der Schaltfläche .

 

Hinweis:

Die Startlochposition muss VOR der Geometrie liegen. Sie kann innerhalb (Matrize) oder außerhalb (Stempel) der Geometrie liegen.

  • Klicken Sie auf die Schaltfläche Wähle Linien/Bögen >>, um festzulegen, aus welchen Geometrieelementen Sie die Bearbeitungsfigur erzeugen wollen.

  • Nach Wahl der Schaltfläche zeigt das System die Standardauswahlabfrage zur Auswahl der Geometrieelemente.

  • Wählen Sie die Option Verbinden (rechte Maustaste od. horizontale Ikonenleiste) und klicken Sie ein Geometrieelement an.

  • Beachten Sie, wie das System alle folgenden Geometrieelemente automatisch auswählt.

 

 

  • Beenden Sie die Auswahl mit der Option Ende (siehe rechte Maustaste od. horizontale Ikonenleiste).

  • Das folgende Bild zeigt die fertige Figur, nebst Startlochposition.

 

 

  • Erzeugen Sie weitere Bearbeitungsfiguren oder beenden Sie den Dialog mit der Schaltfläche OK.
