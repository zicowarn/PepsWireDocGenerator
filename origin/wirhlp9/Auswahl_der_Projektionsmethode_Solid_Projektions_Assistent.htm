Auswahl der Projektionsmethode

Diese Option legen fest, welche Methode zum Ableiten der 2D-Geometrie verwendet werden soll.

Die Ableitung kann automatisch (Option "Erodiergeometrie automatisch erkennen") oder manuell (die anderen Optionen) durch Selektieren einzelner 3D-Geometrien (Faces od. Kanten) erfolgen.

 

Erodiergeometrie automatisch erkennen

Der Solid wird automatisch nach erodierbaren 2D-Geometrien durchsucht.

Die Option Gesamte Zeichnung wird verwendet, wenn alle vorhandenen Solids durchsucht werden sollen.

Ist die Option nicht aktiv, folgt eine Möglichkeit zur Auswahl des(r) Solids der(ie) durchsucht werden sollen.

 

 

Faces verbinden

Bei dieser Methode können Sie ein vertikales Face innerhalb des Solidmodells wählen.

Das System sucht dann automatisch alle mit dem gewählten Face verbundenen Faces und leitet daraus 2D-Geometrien ab.

 

 

Oberes Face

Bei dieser Methode können Sie ein oder mehrere horizontale Faces innerhalb des Solidmodells wählen.

Das System sucht alle erodierbaren Geometrien innerhalb der gewählten Faces und leitet daraus 2D-Geometrien ab.

 

 

Kanten

Bei dieser Methode können Sie Kanten innerhalb des Solidmodells wählen. Das System erzeugt aus den gewählten Kanten 2D-Geometrien.

 

 

Manuelle Auswahl

Bei dieser Methode können Sie ein oder mehrere vertikale Faces innerhalb des Solidmodells wählen. Das System erzeugt 2D-Geometrien aus den gewählten Faces.
