Zusätzliche Anbindungen

Der Befehl Zusätzliche Anbindungen findet Anwendung, wenn an eine Bearbeitungsfigur mehrere Haltestege (Anbindungen) angebracht werden müssen.

Ein einzelner Haltesteg (Anbindung) wird mit der Funktion Stoppunkt od. Anbindung im Dialog Bearbeitung einstellen erzeugt.

Nach Aufruf des Befehls wird der Anwender aufgefordert, die Bearbeitungsfigur zu wählen, an der die Anbindungen (Haltestege) angebracht werden sollen.

Verwenden Sie anschließend die Fangfunktionen, um die Positionen der Startlochbohrungen für die Anbindungen (Haltestege) festzulegen.

Die Länge des Haltestege wird zu diesem Zeitpunkt noch nicht bestimmt. Die Position der Startlochbohrung wird für jede Anbindung (Haltesteg) durch ein kleines blaues Quadrat symbolisiert.

 

 

Die Länge der Anbindungen (Haltestege) wird im Dialog Bearbeitung einstellen im Register Anbinden festgelegt.

Aktivieren Sie dort die Checkbox Zusätzliche Anbindungen, um festzulegen, dass die zusätzlichen Anbindungen bei einer Bearbeitung berücksichtigt werden sollen.

Im Register Anbinden wird ebenfalls festgelegt, wann und auf welche Weise die Haltestege entfernt werden.

 

 

Innerhalb des Befehls stehen drei Optionen zur Verfügung: Steg kopieren + verschieben, Steg kopieren + rotieren und Steg Spiegeln (siehe rechte Maustaste od. horizontale Ikonenleiste).

Die Optionen finden bei symmetrischen Formen Verwendung, die das Kopieren von vorhandenen Haltestegsymbolen erlauben.

 

 

Beachten Sie auch die Option Löschen (siehe rechte Maustaste od. horizontale Ikonenleiste), diese ermöglicht das Löschen von fehlerhaften Haltestegsymbolen.

Zusätzliche Anbindungen (Haltestege) sollten immer VOR der Programmierung der Bearbeitungsstrategie bestimmt werden. Sie können diese aber auch nachträglich bestimmen.

Es ist im so einem Fall aber notwendig, die Bearbeitungsoperation mit dem Befehl Operation editieren zu editieren um die Länge der Anbindung zu bestimmen.

Falls das Drahtmodell der NC-Bearbeitung bei dem oben geschilderten Vorgang vom Bildschirm verschwindet, können Sie dieses mit den Befehl Grafik NC-Bearbeitung neu erzeugen erneut aufbauen.
