Matrizen- / Stempeltyp

Diese Listbox enthält die Geometrieformen, die das System bei der automatischen Erzeugung von Stempel- und Matrizenfiguren erkennt.

Jeder Geometrieform können Sie separate Einstellungen für die Startlochposition zuordnen. Bei der Auswahl einer Geometrieform zeigt das System ein Bitmap das die Form, eine typische Startlochposition und deren Parameter Position grafisch anzeigt.

 

Folgende Geometrieformen existieren:

 

Rund kreisrunde Stempel/Matrizen

Rechteck rechteckige Stempel/Matrizen

Rechteck rund an den vier Ecken verrundete rechteckige Stempel/Matrizen

Langloch Geometrien von Typ Langloch

L-Form L-förmige Stempel/Matrizen

D-Form / DD-Form D- und DD-förmige Stempel/Matrizen

Beliebige Kontur jede andere geschlossene Form

 

Beachten Sie, dass die Einstellungen für jede Form mit der Schaltfläche Einstellungen speichern für spätere PEPS Sitzungen gespeichert werden können.

(Diese Option ist bei Verwendung eines Suchschemas nicht aktiv).
