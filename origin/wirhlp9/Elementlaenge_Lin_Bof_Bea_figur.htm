Elementlänge

Soll das System Geometrieelemente die kleiner sind, als eine bestimmbare Länge, bei der Figurerzeugung farblich kennzeichnen, dann können Sie dies mit den Optionen Zeige Elemente kleiner als und Graphisch in ... einstellen.

Wenn Sie einen Report über kleine Elemente ausgegeben wollen, müssen Sie die Option In Report aktivieren.
