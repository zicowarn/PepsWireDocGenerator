PROJECT THROUGH A SINGLE SHAPE BY A CONSTANT ANGLE

 

 

Pick the Select Profile>> Button and choose the First Profile Figure.

The Red Exclamation Mark changes to a Green Tick indicating that a profile has been selected.

 

 

Pick the Next Button, to move on.

 

Type in the required Angle.

Select the required Taper Orientation Button.

 

 

Upper Projection

Tick/Enable and enter a Height for it to be produced.

 

Lower Projection

Tick/Enable and enter a Height for it to be produced.

 

Equalise sections with equal span count

Tick/Enable for the two Profiles will be linked using proportional linear equalisation if not enabled the corresponding span ends of the two will be linked

 

Produce Figures

Tick/Enable to Produce Figures instead of Line/Arcs.

 

Figure Name Prefix

Will be required if Figures are to be produced.

 

Projection Tolerance

Controls how the Lines/Arcs are vectored when generating the projected geometry.

The lower the value the more accurate the projected geometry is.

 

Pick the Project Button, to produce required Projection(s).

 

 

Pick the Finish Button.
