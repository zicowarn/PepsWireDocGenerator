Erzeuge Rohteil vom Solid-Body

Es wird automatisch eine Rohteilgeometrie erzeugt, die von dem zu bearbeitenden 3D-Modell abgeleitet wird. Das manuelle Erzeugen eines Rohteils entfällt.

 

Bearbeitet Konik als Regelflächen

Wird vom 3D-Modell eine umlaufende oder variable Konik abgeleitet, dann wird diese Konik als 4-Achsen Regelfläche an PEPS übergeben.
Ohne Aktivieren der Option wird die Konik, als umlaufende Konik oder Variable Konik an PEPS übergeben.

 

Regelflächen in Rotationsachsen umwandeln –

Wird vom Modell eine 4-Achsen Regelfläche abgeleitet, dann wird diese als Regelfläche mit Rotationsachse an PEPS übergeben.

Auf diese Weise kann eine Regelfläche mit Hilfe einer oder zwei zusätzlichen Rotationsachsen bearbeitet werden (optional).
