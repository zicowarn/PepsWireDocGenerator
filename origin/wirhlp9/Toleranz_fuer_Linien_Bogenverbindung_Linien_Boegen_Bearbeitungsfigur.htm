Toleranz für Linien/Bogenverbindung

In diesem Textfeld stellt man einen Korrekturwert für die Endpunktverbindung ein.

Die Option wird normalerweise nur dann gesetzt oder verändert, wenn importierte 2D-CAD-Daten kleinste Lücken aufweisen.

Der eingegebene Wert legt fest, dass Lücken die kleiner oder gleich dem Wert sind, automatisch geschlossen werden.

Das Feld muss immer einen Wert größer als Null (0) (z.B. 0.001) enthalten und wird vom Feld Geometrie Toleranz des Dialogs Geometrie Optionen im Menü Datei \ Einstellungen \ Systemeinstellungen voreingestellt.

 

Die Qualität von CAD Daten kann auch mit dem Befehl Zeichnung reparieren im Menü Ändern > Hilfsfunktionen beeinflusst werden.
