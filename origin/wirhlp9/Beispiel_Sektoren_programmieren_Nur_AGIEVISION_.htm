Sektoren programmieren (Nur AGIEVISION)

Ausgangssituation: Es ist eine Bearbeitungsfigur vorhanden. Diese soll in einem durch 2 Punkte definierten Sektor mit einer zusätzliche Schnittluft versehen werden.

 

 

  • Rufen Sie im Menü Vorbereitung den Befehl Sektoren auf.

Wähle Figur :

  • Klicken Sie die Bearbeitungsfigur an, die mit einem Sektor versehen werden soll.

Eingabe Punkt für Sektorstart:

  • Geben Sie die Position des Sektorstarts ein oder verwenden Sie die Fangfunktionen.

Ein Richtungspfeil zeigt an, in welcher Richtung der 2te Punkte, das Sektorende definiert werden muss.

Sie können die Richtung ändern mit Hilfe der Option Richtung tauschen (siehe rechte Maustaste od. horizontale Ikonenleiste).

 

 

Eingabe Punkt für Sektorende:

  • Geben Sie die Position des Sektorendes ein oder verwenden Sie die Fangfunktionen.

  • Es folgt der Sektoren-Dialog in dem man die gewünschten Sektorparameter eingibt.

 

 

  • Beenden Sie den Dialog mit der Schaltfläche OK.

  • Der Sektorbereich wird durch zwei hellblaue Symbole gekennzeichnet.

 

  • Beenden Sie den Befehl mit der Option Ende (siehe rechte Maustaste od. horizontale Ikonenleiste).
