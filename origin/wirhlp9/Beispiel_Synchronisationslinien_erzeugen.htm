Synchronisationslinien erzeugen

Ausgangssituation: Es sind zwei Bearbeitungsfiguren für eine Regelflächenbearbeitung vorhanden. Diese sollen durch Synchronisationslinien für eine Regelflächenbearbeitung vorbereitet werden.

Ohne Synchronisationslinien würde es zu einem verzerrten Bearbeitungsergebnis kommen.

 

 

  • Rufen Sie im Menü Vorbereitung den Befehl Synchronisationslinien auf.

Wähle UNTERE Ebene :

  • Klicken Sie die Bearbeitungsfigur für die untere Regelflächenebene an.

Wähle OBERE Ebene :

  • Klicken Sie die Bearbeitungsfigur für die obere Regelflächenebene an.

Eingabe Beginn der Synchronisationslinie :

  • Bestimmen Sie den Anfang der ersten Synchronisationslinie mit der Fangfunktion Endpunkt fangen.

Eingabe Beginn der Synchronisationslinie :

  • Bestimmen Sie das Ende der ersten Synchronisationslinie mit der Fangfunktion Lot fangen.

 

 

Eingabe Beginn der Synchronisationslinie :

  • Bestimmen Sie die restlichen drei Synchronisationslinien auf die gleiche Art und Weise.

 

 

Tip: Beachten Sie auch die Optionen im Kontextmenü auf der rechten Maustaste. Sie können damit vorhandene Synchronisationslinien löschen, kopieren, spiegeln usw. .

 

  • Beenden Sie den Befehl mit der Option Ende (siehe rechte Maustaste od. horizontale Ikonenleiste).

  • Erzeugen Sie eine Regelflächenbearbeitung und aktivieren Sie im Dialog die Checkbox Synchronisationslinien.

  • Simulieren Sie die Bearbeitung z.B. mit der Solid-Simulation.

 

Hinweis:

Sie können Synchronisationslinien durch einen Doppelklick auf die roten Synchronisationslinien editieren und nachträglich verändern.
