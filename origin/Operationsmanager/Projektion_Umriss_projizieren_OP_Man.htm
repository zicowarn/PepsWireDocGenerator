Projektion

 

Dieser Dialogbereich wird verwendet, um zusätzlich die Projektionsposition der Umrissgeometrie festzulegen.

Die Option 2D-Projizieren wird verwendet, um die drei XYZ-Eingabefelder freizuschalten.

Sind die Felder "ausgegraut" oder die Werte auf 0 gesetzt, wird die Umrissgeometrie wie im Feld Blickrichtung beschrieben, projiziert.

Wenn das Feld Y im Dialogbereich Blickrichtung eine 1 enthält, dann müssen Sie im Dialogbereich Projektion die Projektionsposition ebenfalls im Feld Y bestimmen und die anderen Felder mit 0 belegen.

Das folgende Bild zeigt drei Umrissgeometrien (XY, ZY, ZX), die durch Eingabe von Werten in den Feldern Projektion, verschoben wurden.

 

 

Das gleiche Beispiel OHNE Eingabe von Projektionswerten finden Sie siehe Blickrichtung.
