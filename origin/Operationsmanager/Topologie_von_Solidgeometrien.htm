Topologie von Solidgeometrien

 

Sheet-Body

Ein Sheet-Body besteht immer aus einem oder mehreren Faces. Jedes Face innerhalb eines Sheet-Bodies besteht aus einem äußerem Loop und kann innere Loops enthalten.

Jedes Loop besteht aus einer oder mehreren Kanten. Jede Kante ist ein einzelnes Geometrieelement z.B. ein Bogen, eine Linie oder eine NURBS-Kurve.

 

Solid-Body

Ein Solid-Body besteht immer aus einer Anzahl von Faces. Diese Faces können wiederum in einzelne Geometrien aufgelöst werden (siehe Sheet-Body oberhalb).

 

Draht-Body

Ein Draht-Body kann offen oder geschlossen sein und besteht aus einer oder mehreren Kanten.
