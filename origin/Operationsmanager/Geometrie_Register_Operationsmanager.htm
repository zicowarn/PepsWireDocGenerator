Geometrie Register (Operationsmanager)

 

Das Register Geometrie wird verwendet, um alle Solidgeometrieelemente aufzulisten (Solids, Sheet-Bodies, Draht-Bodies und Baugruppen) die erzeugt, importiert oder abgeleitet wurden.

Sie können die gezeigt Liste verwenden, um eine Solidgeometrie zu identifizieren. Klickt man auf ein Symbol für eine Solidgeometrie, dann zeigt das System die gewählte Geometrie hervorgehoben im Grafikfenster an.

 

 

Ein Klick mit der rechten Maustaste auf einen Eintrag im Operationsmanager, führt zum Aufruf eines Kontextmenüs.

Die Optionen Löschen, Umbenennen und Eigenschaften dienen zum Löschen, Umbenennen oder Anzeigen der Eigenschaften (Schwerpunkt, Fläche, Masse ....) der gewählten Solidgeometrie.

Da es aber oftmals nötig ist, z.B. die Kanten einer Solidgeometrie abzuleiten, verfügt das System im Kontextmenü über eine Konvertierroutine (Option Konvertieren), mit der man die benötigten Geometrieelemente ableiten kann.

Siehe Befehl Geometrien konvertieren im Menü Ändern.

Nähere Informationen über eine Solid-Geometrie kann ebenfalls ermitteln, indem man die Maus über eine Solid-Geometrie positioniert und dann die rechte Maustaste klickt.
