Eigenschaften

 

Die Eigenschaften einer Solid-Geometrie kann man ermitteln, indem man die Maus über eine Solid-Geometrie positioniert und dann die rechte Maustaste klickt. Man kann die Eigenschaften ebenfalls im Operationsmanager anzeigen lassen, indem man dort den Eintrag einer Solid-Geometrie anklickt und dann die rechte Maustaste klickt.

Die gezeigte Information ändert sich, je nachdem welches Solid-Element man anklickt. Das folgende Beispiel zeigt die Eigenschaften eines Solid-Bodies.

 

 

Beachten Sie, dass Sie die gleichen Informationen mit Hilfe des Befehls Element Info erhalten können.
