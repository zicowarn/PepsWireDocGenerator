Kopieren

 

Die Option Kopieren erlaubt das Kopieren einer oder mehrerer Operationen aus der Operationsliste.

Anschließend könnten Sie die kopierte Operation(en) mit der Option Einfügen an anderer Stelle des Operationsmanagers einfügen.

Verwenden Sie zur Auswahl der Operationen die Standard MS Windows Auswahltechniken.

 

So wird es gemacht:

Bearbeitungsoperation kopieren
