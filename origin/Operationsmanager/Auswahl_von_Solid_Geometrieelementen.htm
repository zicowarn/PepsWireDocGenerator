Auswahl von Elementen

Ruft man den Befehl Element Info auf und setzt man VOR der Geometrieauswahl keinen Filter, zeigt das System nur generelle Informationen über Solid-Bodies, Sheet-Bodies oder Draht-Bodies) an.

Benötigte man Informationen über die Bestandteile von Solids (Kanten, Loops oder Faces), dann sollten Sie die Option Filter (siehe rechte Maustaste) verwenden und den Elementtyp einstellen über den Sie Informationen benötigen.
