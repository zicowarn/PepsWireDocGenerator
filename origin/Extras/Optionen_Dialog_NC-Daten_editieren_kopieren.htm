Optionen

 

In diesem Dialog können Sie den Dialog NC-Daten editieren / kopieren voreinstellen.

 

Ziellaufwerk: Hier können Sie einstellen, in welches Verzeichnis die Daten kopiert werden sollen, wenn Sie die Option Kopiere Datei auf Diskette verwenden.

 

DNC-Aufruf: In diesem Feld können Sie den Pfad zur Datenübertragungssoftware und den Namen des Übertragungsprogramms eingeben.

Normalerweise ist dies die PEPS Software WNCCOMMS32.EXE, die im PEPS Unterverzeichnis ...\System abgelegt ist (Optionale Dienstleistung).

 

Editor-Software: Hier können Sie einstellen, welcher Texteditor zum Editieren der NC-Dateien verwendet werden soll.

Die Voreinstellung ist das Programm MS Windows Notepad.exe.

 

Software für Simulation: In diesem Feld können Sie den Pfad zu einer externen Simulationssoftware eingeben.

Ziellaufwerk 2: Hier können Sie ein zweites, alternatives Laufwerk (z.B. für einen USB-Stick) einstellen, in das die Daten kopiert werden sollen, wenn Sie die Schaltfläche Ziellaufwerk 2 klicken.

 

Kopiere Einrichteblatt: Ist diese Option aktiv, wird neben den NC-Daten, auch das Einrichteblatt in das Ziellaufwerk kopiert.
