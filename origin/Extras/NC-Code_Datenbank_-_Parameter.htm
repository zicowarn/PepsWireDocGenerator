NC-Code Datenbank - Parameter

 

Speichernamen

Die Liste zeigt alle aktuell vorhandenen NC-Codes an.

Die 4 Schaltflächen unterhalb der Liste dienen zum Navigieren in der NC-Codeliste.

 

Name zum Laden / Speichern / Löschen

Zeigt den aktuell in der Liste gewählten NC-Code an.

 

Bemerkung

Zeigt die Bedeutung des aktuell gewählten NC-Codes an.

 

Funktionsschaltflächen

Die Schaltflächen haben folgende Bedeutung:

  : Suchen in der NC-Code Datenbank.

  : Neuen NC-Code zur Liste hinzufügen

  : Gewählten NC-Code in die MS-Windows Zwischenablage kopieren.

  :  Gewählten NC-Code verändern.

  : Gewählten NC-Code löschen.

  : Ruft einen Einstell-Dialog auf mit verschiedenen Einstellparametern:
