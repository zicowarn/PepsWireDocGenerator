Grafisches Einrichteblatt

 

Neben dem Einrichteblatt, dass automatisch bei einem Postprozessorlauf (NC-Daten erzeugen) erzeugt wird, existiert ein weiteres, das "Grafische Einrichteblatt".

Das "Grafische Einrichteblatt" ist eine interaktive Art, ein Einrichteblatt zu erzeugen. Die im Einrichteblatt anzuzeigenden Geometrieelemente werden dabei interaktiv ausgewählt.

Zudem erzeugt das System automatisch einen Zeichnungsrahmen und schreibt Informationen (z.B. Material, Materialhöhe, Drahtdurchmesser) in den Kopf des Einrichteblatts.

 

Wichtiger Hinweis:

Dieser Befehl kann nur ausgeführt werden, wenn ein Postprozessorlauf (NC-Daten erzeugen) ausgeführt wurde.

Vorgehensweise:

  • Programmieren Sie das  Werkstück und erzeugen Sie das NC-Programm.

  • Wählen Sie aus dem Menü Extras die Funktion Grafisches Einrichteblatt.

  • Das System fragt:

Wähle Objekte ... :

  • Wählen Sie die Geometrieelemente aus, die im Einrichteblatt angezeigt werden sollen.

  • Beenden Sie die Auswahl mit der Option Ende (siehe rechte Maustaste od. horizontale Ikonenleiste).

Es wird ein Dialog mit Eintellparametern für das Einrichteblatt angezeigt:. Stellen Sie die Dialogparameter wie benötigt ein.

Wichtiger Hinweis:

Bei Erstanwendung müssen Sie die Schaltfläche Standardwerte laden verwenden, um den Dialog voreinzustellen.

 

Klicken Sie auf die einzelnen Bereiche des folgenden Bildes, um Hilfetexte aufzurufen.

  • Beenden Sie den Dialog mit OK.

  • Das System erzeugt das Einrichteblatt und zeigt es anschließend im Grafikfenster an.

  • Das System fragt:

Wähle benötigte Option:

  • Klicken Sie nun die rechte Maustaste und wählen Sie im Kontextmenü Drucke.

 

Das Einrichteblatt wird  ausgedruckt.

  • Beenden Sie die Druckfunktion mit der Option Ende (siehe rechte Maustaste od. horizontale Ikonenleiste).
