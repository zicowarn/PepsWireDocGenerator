NC-Daten

 

Mit diesen Ikonen können Sie die NC-Datendateien des momentanen Programmes anzeigen/editieren, ausdrucken und auf einen Datenträger kopieren (z.B. eine Diskette), ohne den Windows Explorer aufrufen zu müssen.

Kopiere Datei auf Zielverzeichnis

Diese Schaltfläche kopiert die NC-Datendatei des momentanen Programms in ein bestimmbares Laufwerk / Verzeichnis – Einstellbar mit der Schaltfläche Optionen).

 

Zeige Datei

Mit dieser Schaltfläche können Sie die NC-Daten des momentanen Programmes in einem Editor aufrufen. Welcher Editor (Voreinstellung = Windows Notepad Editor) verwendet wird, können Sie mit der Schaltfläche Optionen einstellen.

 

Zeige Einrichteblatt

Mit dieser Schaltfläche können Sie die Einrichteblattdatei (PDF-Datei) des momentanen Programms über einen Viewer anzeigen.

 

Kopiere Datei auf Zielverzeichnis 2

Diese Schaltfläche kopiert die NC-Datendatei des momentanen Programms auf ein zweites, alternatives Laufwerk (Einstellbar mit der Schaltfläche Optionen).

 

 

NC-Code Simulation

Diese Schaltfläche startet die NC-Code Simulation.
