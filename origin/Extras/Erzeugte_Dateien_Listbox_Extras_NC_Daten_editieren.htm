Erzeugte Dateien Listbox

 

Es gibt Maschinen (z.B. ACVision), für die mehr als eine NC-Datei erzeugt wird.

Ist dies der Fall, wird der Dialog NC-Daten editieren / kopieren um eine Listbox erweitert.

Auf diese Weise können Sie auswählen, welche Datei bei Wahl der Schaltfläche Zeige Datei angezeigt wird.
