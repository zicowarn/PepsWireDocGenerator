Operation starten ... (AGIEVISION)

Dieser Dialog ist immer die erste Bearbeitungsanweisung. In dem Dialog werden der Name der NC-Ausgabedatei und andere Parameter festgelegt, die für die NC-Daten benötigt werden.

Falls der Programmstart Dialog noch nicht ausgefüllt wurde, ruft das System diese Box automatisch auf und erst anschließend den Dialog Operation starten.

Jede neue Operation führt zu einem weiteren Eintrag in der Operationsliste und zu einer neuen NC-Ausgabedatei.

Auf diese Weise können Sie mit einem Programm mehrere NC-Ausgabedateien erzeugen.
