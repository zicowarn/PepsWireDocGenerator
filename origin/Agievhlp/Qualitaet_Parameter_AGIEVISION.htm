Qualität / Parameter

Mit den Einstellungen in diesem Dialog legt man hauptsächlich das Qualitätsziel fest.

Aufgrund der Angaben für das Qualitätsziel im Register Qualität berechnet die AGIEVISION Steuerung die Anzahl der Schnitte, die nötigen Offsetwerte und Generatoreinstellungen.

 

Im Register Schnittluft/Ecken kann man Angaben über die Größe von zusätzlicher Schnittluft/Trennspiel eingeben, sowie das Verhalten des Drahtes an Innen / Außenecken einer Kontur festlegen.
