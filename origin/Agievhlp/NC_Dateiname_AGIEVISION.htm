NC-Dateiname

Legt den Namen der NC-Ausgabedatei(en) fest. Als Vorgabe zeigt das System in diesem Feld den Dateinamen des momentanen Programms.

Es wird für jede geschnittene Bearbeitungsfigur eine separate NC-Datei erzeugt. Der Name der Dateien wird automatisch inkrementiert (Test.iso, Test1.iso, Test2.iso, Test3.iso ...).
