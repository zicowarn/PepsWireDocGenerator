Einstellungen (Einstellen der Schnittmethode AGIEVISION)

Die Schaltfläche Einstellungen wird verwendet um den Dialog Bearbeitung einstellen aufzurufen.

In diesem Dialog kann man für eine Konturbearbeitung die Parameter für Schnitte. Anbindungen, Ein/Ausfahren, Offset, Datenbank, Farben einstellen.

 

Klicken Sie auf die einzelnen Bereiche des folgenden Bildes, um Hilfetexte aufzurufen.
