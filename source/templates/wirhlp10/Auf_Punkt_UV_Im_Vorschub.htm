准点（紫外线）

此字段中注明了 送丝过程中 焊丝应移动到的 UV 位置 。

仅当发生倾斜进 给 运动 （倾斜焊丝位置） 时才需要 UV 规格 。

UV位置 直接 输入到字段中。 指定 目标位置的 增量 U 和 V坐标。

或者，可以使用UV >> 按钮从现有 CAD 绘图中确定 UV 目标位置。

在这种情况下，请使用 端点坐标 捕捉功能。

该对话框将增量 UV 位置输入与当前 XY 输入相关联。

因此，必须在 UV 位置信息 之前对 XY 位置进行编程。

对于纯 2 轴运动， 无需 对 UV 位置进行编程 。
