示例：更改处理大量图形的操作的切割顺序。

  • 使用Operations Manager 编辑现有编辑操作 。 该操作应包含 3 个以上的数字，如下例所示。

  • 单击 “重新排序” 按钮。 系统显示 “优化” 对话框。

当前切割顺序由小箭头符号指示。 下图可以看出，顺序相当混乱。

现在应该更改切割顺序，使其从左下角开始 并沿 X 轴 逐行 运行。

  • 接下来，使用 按钮选择 要剪切的第一个图形的起点（左下图）。

  • 然后将 优先级 字段中的 滑块 移至 X 。

  • 然后单击 优化 按钮。

下图显示了优化后的切割顺序。

  • 使用“关闭”按钮 关闭 “优化” 对话框 。
