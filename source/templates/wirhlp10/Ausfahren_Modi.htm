伸直

导致线性（直线）伸展运动。

使用拐角模式延伸圆弧

导致加工图形 发生 圆形切向延伸运动。

对于此选项，您必须编程一个扩展半径。

延伸运动的半径必须小于起始（孔）位置与图形上加工起始位置之间的距离，并且大于导线半径加上偏移量。

循环扩展有两种方法，即：

使用 ISO 延伸圆弧 和 使用拐角模式延伸圆弧

仅当加工锥度时，这些方法之间的 差异 才 变得明显 。

通常，您使用扩展选项 extend arc with ISO 。 如果您想以弧线移动到受控曲面或从受控曲面移出，这也是唯一可能的方法。

下图显示了 使用 ISO 延伸圆弧的 方法。

如果 激活使用拐角模式延伸圆弧 选项，则会创建 ISO 半径或车削主轴半径，具体取决于 标准轮廓处理 对话框中的 拐角模式设置。

“使用角模式延伸拱形” 选项与 “使用 ISO 延伸拱形 ”选项 不同 。

使用 ISO 角模式扩展圆弧

延伸圆弧：旋转主轴转角模式

仅当 NC 控制允许时，才应对 上面显示的扩展方法（ ISO 和 车削主轴）进行编程。

如果有疑问，请使用 “Extend Arc with ISO” 选项。

另请注意，半径元素上的弧形接近/延伸会导致许多控制中的偏移误差。

因此，最好在直线元件上驱动开/关。

使用 ISO 延长弓

导致弧形伸展运动。

线材在编程半径内远离加工图形。 如果对圆锥体进行编程 ，则延伸运动始终形成 ISO 半径 。

必须指定 延伸运动 的 半径 。 在半径 0（零）处，车辆当前正在远离。
