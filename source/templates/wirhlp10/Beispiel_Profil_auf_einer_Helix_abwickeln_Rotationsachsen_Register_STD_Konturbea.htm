在螺旋上展开轮廓

待切割轮廓以 2D 方式切割，毛坯绕旋转轴旋转。 轮廓在柔和的曲线上呈螺旋状发展。

这意味着旋转轴与切割喷嘴同时沿着旋转轴移动。

下图显示了在旋转轴同时旋转的情况下对细长孔进行圆柱加工。

如果 激活螺旋上的展开轮廓选项 ，系统将解锁以下选项：

旋转，每毫米度数

旋转 ，每毫米度数 参数定义每毫米切割的旋转轴旋转（以度为单位）。

螺旋线起点的 X 位置

该 X 位置坐标允许切削几何形状沿 X 轴移动。 然后螺旋形状从定义的 X 位置开始。
