变量二次曲线的示例程序

下图示例显示了 要使用 可变锥度功能加工的对称工件。

加工图形按顺时针方向定义。 较小的轮廓是较低的水平，这意味着顶部导丝器必须向外偏转。

因此，我们输入 负锥角（参见 标准 锥度 的说明 ）。

编程变量 锥度（示例程序）：

  • 从“编辑” 菜单中，选择 “标准轮廓编辑” 命令 。

  • 使用 >> 按钮确定 编辑图形的名称。

  • 在“角度”字段中 输入 0 度 作为 起始角度。

  • 在“角模式”框中 ， 选择 “ISO + 锐度” 。

  • 单击 设置 按钮显示 编辑设置 对话框 ：

  • 单击 “绑定” 选项卡 并定义 3 毫米的桥 。

  • 单击电源 开/关 选项卡：

  • 确定一条直线作为 进场/出场策略 。

  • 单击 偏移 选项卡：

  • 在内部轮廓（模具） 字段中，激活 右侧（顺时针 ） 选项 。

  • 关闭 偏移 对话框。 系统切换回 标准轮廓处理 对话框。 从那里，单击 变量 锥度 按钮。

由于起始角度设置为 0 度，因此屏幕图形中不会显示锥度。

选择元素附近的位置：

  • 激活 坐标 捕捉功能 ，然后用鼠标单击半径 1 毫米后的 45 度线（参见图中的位置 1）。

  • 所选择的线是锥度应变为 2 度的线。

  • 在对话框中，单击图像 并指定 2 度作为 结束角度 。 边角模式为 ISO+锐利 。

屏幕图形发生变化并显示编程的锥度。

选择元素附近的位置：

  • 现在图形化右下侧半径 10 毫米中间的一个点（参见图中的位置 2）。

  • 在对话框中，指定 6 度作为 结束角度 。 边角模式依然是 ISO+Sharp 。

  • 请注意屏幕图形中锥度的变化。

由于我们的示例几何形状 是 对称的 ，因此可以 镜像 角度变化符号 。

选择元素附近的位置：

  • 选择 镜像 选项（参见 鼠标右键 或 水平图标栏 ）。

选择对象 o. S=选择 D=删除 ................................ :

  • 如有必要，放大并用鼠标选择两个角度变化符号。

  • 使用“结束” 选项退出选择模式 （请参见鼠标右键或水平图标栏）。

输入镜像轴的第一个点：

  • 使用交点 捕捉功能，捕捉 到穿过 X0 的垂直 C 线上的交点（参见绘图）。

输入镜像轴的第二个点：

  • 将另一个交点 捕捉 到穿过 X0 的垂直 C 线（见图）。

接下来是一个对话，问题 是应该保留原件吗？ 。 单击 是 按钮。

  • 使用“结束” 选项结束命令 （ 鼠标右键 或 水平图标栏 ）

变量 锥度 的命令输入现已完成。 屏幕显示应类似于上面的示例图。

请注意，在编程过程中， 角度变化点 显示 为 红色小圆圈符号。

通过删除角度变化符号可以取消不正确的输入。 为此，请使用 “删除” 选项。

另请注意 Var 选项。 编辑角 （ 鼠标右键 或 水平图标栏 ）允许您更正不正确的 开始/结束角度 信息或更改 角类型 。
