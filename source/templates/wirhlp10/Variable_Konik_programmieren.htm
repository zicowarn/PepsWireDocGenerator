程序变量二次曲线

编程变量 锥度（两个选项）。

1.) 首先确定整个零件周围相同的锥度。

然后，使用 “可变锥度” 选项，将不同的锥度分配给锥度偏离周围锥度的几何元素。

2.) 或者，首先在“角度”字段中将锥角设置为 0 度 ， 然后选择 “可变锥度” 按钮以确定锥度发生变化的元素。

选择 变量 锥度 按钮后，输入的角度将分配给整个加工图形，系统查询如下：

选择元素附近的位置：

  • 现在确定几何元素中心附近 应更改锥度的 位置。

  • 使用 中心 或 坐标 捕捉来确定位置。

选择元素后，所选元素的中心会标有一个小十字，并 调用 “Variable 锥度”对话框。

单击下图的各个区域可访问帮助文本。

系统在 鼠标右键 或 水平图标栏中 显示更多选项 ：

单击下图的各个区域可访问帮助文本。

  • 单击“角度方向” 框中的图形之一 以设置锥度方向。

  • 现在在对话框中 输入 结束角度（新角度）。 接下来，选择 您需要的 转角模式 。

  • “ 确定” 按钮执行输入并将 角度变化符号 （红色圆圈）插入图形中。

系统自动将角度变化符号放置在所选元素的中心。

可以使用删除 选项 删除 符号（和关联的锥度） 。

  • 重复该过程，直到定义了所有锥度更改。

执行该命令时，可以使用 “复制+移动” 、 “复制+旋转” 和 “镜像” 选项 （请参见 鼠标右键 或 水平图标栏）。

该选项用于复制角度变化符号。
