{% trans "@ Zusätzliche Technologien" %}

{% trans "Der Befehl [Zusätzliche] [Technologien] findet Anwendung, wenn in eine Bearbeitungsfigur zusätzliche Technologieinformationen eingebracht werden müssen." %}

{% trans "Wenn die Figur nur mit einer globalen Technologie pro Schnitt geschnitten oder wenn nur die Start/Endtechnologie bestimmt werden muss, verwendet man den Befehl [Zusätzliche] [Technologien] nicht." %}

{% trans "Im Gegensatz zur Definition von Start/Endtechnologien, wo der Technologiewechsel nur am Anfang bzw. Ende einer Bearbeitungsfigur stattfindet, erlaubt der Befehl [Zusätzliche] [Technologien] das Einfügen von beliebig vielen [Technologiewechseln, an jeder beliebigen Stelle] der Figur." %}

{% trans "Nach Aufruf des Befehls wird der Anwender aufgefordert, die Bearbeitungsfigur zu wählen, an der die Technologiewechsel eingebracht werden sollen." %}

{% trans "Verwenden Sie anschließend die [Fangfunktionen], um die Positionen der Technologiewechselpunkte festzulegen." %}

{% trans "Nach der Eingabe der Position eines Technologiewechselpunktes, erfolgt der Aufruf eines Dialogs in dem die neuen Technologien z.B. Offsetregister oder Stromstufen zugewiesen werden." %}

{% trans "Jede Technologiewechsel wird durch ein kleines Kreissymbol in der Farbe [Magenta] symbolisiert." %}

{% trans "Der Dialog für einen Technologiewechsel ist CNC-Maschinen und CNC-Steuerungsabhängig." %}

{% trans "Der folgende Dialog soll als Beispiel dienen und erlaubt die Eingabe von Stromstufe und Offsetregister:" %}

 

{% trans "[Klicken Sie auf die einzelnen Bereiche des folgenden Bildes, um Hilfetexte aufzurufen.]" %}

 

{% trans "[Hinweis]: Dieser Dialog variiert je nach gewählter CNC-Maschine." %}

 

{% trans "Innerhalb des Befehls stehen drei Optionen für Transformationen zur Verfügung:" %}

{% trans "[Technologie kopieren + verschieben], [Technologie kopieren + rotieren] und [Technologie spiegeln] (siehe [rechte Maustaste] od. [horizontale Ikonenleiste])." %}

{% trans "Die Optionen finden bei symmetrischen Formen Verwendung, die das Kopieren von vorhandenen Technologiesymbolen erlauben." %}

 

 

{% trans "Beachten Sie auch die Option [Löschen], diese ermöglicht das Löschen von fehlerhaften Technologiesymbolen." %}
