{% trans "@ Variable Bearbeitungsebenen" %}

{% trans "Bei Werkstücken mit unterschiedlichen Höhen entsteht bei der Bearbeitung mit Konik die Schwierigkeit, dass bei jeder sich ergebenden Höhenänderung, Referenz- und Sekundärhöhe neu angeglichen werden müssen." %}

{% trans "Eine Möglichkeit diese Problemstellung zu lösen, ist die Projektion des vorhandenen Höhenprofils auf die zu bearbeitende Kontur." %}

{% trans "Da die Erstellung solcher projizierten Konturen bei entsprechenden Höhenprofilen sehr komplex sein kann, ist eine manuelle Erstellung nur bei einfachen Anwendungsfällen ratsam." %}

{% trans "Für darüber hinausgehende Anwendungsgebiete wurde der Befehl [Variable Bearbeitungsebenen] entwickelt, der die Erstellung der projizierten Kontur nach Eingabe der Höhenwerte automatisch übernimmt." %}

 

 

{% trans "Nach Aufruf des Befehls werden Sie aufgefordert die Bearbeitungsfigur zu wählen, auf der die [Höhenänderungen] eingebracht werden sollen." %}

{% trans "Verwenden Sie die Fangfunktionen, um die Positionen der [Höhenänderung] festzulegen." %}

{% trans "Nach der Eingabe der Position erfolgt der Aufruf eines Dialogs in dem Sie die [Bezugsebene] und die [Höhenänderung] eingeben." %}

 

{% trans "[Klicken Sie auf die einzelnen Bereiche des folgenden Bildes, um Hilfetexte aufzurufen.]" %}

 

{% trans "Bestimmen Sie im Dialog die Z-Höhe. Jede Höhenänderung wird durch ein [kleines Dreieck] in der Farbe [Magenta] symbolisiert." %}

{% trans "Fahren Sie fort, bis alle Höhenänderungen bestimmt sind und bearbeiten Sie die Figuren als [Regelfläche]." %}

 

{% trans "Innerhalb des Befehls stehen Optionen für Transformationen zur Verfügung:" %}

{% trans "[Var. Höhen kopieren + verschieben], [Var. Höhen kopieren + rotieren] und [Var. Höhen spiegeln] (siehe [rechte Maustaste] od. [horizontale Ikonenleiste])." %}

{% trans "Die Optionen finden bei symmetrischen Formen Verwendung, die das Kopieren von vorhandenen Symbolen für variable Bearbeitungsebenen erlauben." %}

 

 

{% trans "Beachten Sie auch die Option [Löschen] (siehe [rechte Maustaste] od. [horizontale Ikonenleiste]) die das Löschen von fehlerhaften Symbolen erlaubt." %}

{% trans "Mit der Option [Var. Höhen editieren] können Sie vorhandene Symbole editieren, um die Höhe der Ebene zu verändern." %}
