搜索同直径孔并自动生成加工图形

  • 画出孔。

  • 在“准备\杂项” 菜单中，调用 “ 圆/弧==>多个图形” 命令。

  • 第一个查询： 选择圆、圆弧或组来创建组， 自动确定孔的半径。

  • 单击要确定其半径的孔。

  • 现在选择 “搜索” 选项（请参见 鼠标右键 或 水平图标栏 ）。

  • 系统要求 选择半径<10>： 并在方括号中显示确定的默认半径。

  • 使用 RETURN 按钮接受系统默认值。

  • 接下来是标准选择查询 选择对象....

  • 使用鼠标选择要组合的孔，或使用选择窗口。

  • 单击 “结束” 选项（请参见 鼠标右键 或 水平图标栏 ）。

  • 再次选择 “结束” 选项（请参见 鼠标右键 或 水平图标栏 ）。

  • 现在单击 “孔” 选项（请参见 鼠标右键 或 水平图标栏 ）。

  • 系统要求 孔直径<20>： 并在方括号中显示默认孔的直径。

  • 使用 RETURN 按钮接受系统默认值。

  • 系统现在自动创建一组相同的加工图形。
