{% trans "@ Sekundärfigur (TWIST) erzeugen" %}

{% trans "Dieser Befehl erzeugt eine Kopie (Sekundärfigur) von einer vorhandenen Figur. Anschließend besteht die Möglichkeit, die kopierte Sekundärfigur zu rotieren, transformieren und zu skalieren." %}

{% trans "Mit Hilfe dieses Befehls können Sie auf einfache Weise die z.B. untere Regelflächengeometrie für Schrägschieber oder verdrehte Schrägschieber (TWIST) erzeugen." %}

{% trans "Schrägschieber mit TWIST erzeugen und bearbeiten:" %}

{% trans "In eine Schnittplatte soll eine 20 Grad schräge Bohrung (Schrägschieber) eingebracht werden." %}

 

  • {% trans "Zeichnen Sie den 20 mm Kreis und erzeugen Sie eine Bearbeitungsfigur." %}

  • {% trans "Da wir eine rotierte Figur nicht schneiden können, müssen wir diese auf die XY-Ebene projizieren." %}

  • {% trans "Der Schrägschieber wird als Regelfläche geschnitten, deshalb benötigen wir eine weitere, verschobene Figur für die untere (z-20) Regelfläche." %}

  • {% trans "Der Befehl [Sekundärfigur (TWIST) erzeugen] erzeugt eben diese zweite Figur (Sekundärfigur)." %}

  • {% trans "Wählen Sie den Befehl [Sekundärfigur (TWIST) erzeugen] im Menü [Vorbereitung \ Diverses]." %}

{% trans "Startposition od. Mittelpunkt für die Skalierung/Rotation/Spiegelung :" %}
