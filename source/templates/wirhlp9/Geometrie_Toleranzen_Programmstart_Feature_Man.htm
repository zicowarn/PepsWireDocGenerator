{% trans "Geometrie Toleranzen" %}

{% trans "Bei Verwendung der Funktionen #Automatische Feature-Erkennung# und #Manuelle Feature-Erkennen# , führt das System eine Untersuchung der zu bearbeitenden Flächen und Kanten durch. Hier wird überprüft, ob die Flächen und Kanten erodierbar sind oder nicht." %}

{% trans "Im Dialogbereich [Geometrie Toleranzen] stellen Sie ein, mit welcher Genauigkeit die Flächen und Kanten eines Modells untersucht werden." %}

{% trans "Flächenübergänge und Kanten die außerhalb der eingestellten Toleranzen liegen, werden als nicht erodierbar eingestuft und folglich werden für diese Elemente auch keine Bearbeitungsfeatures erzeugt." %}

{% trans "Durch Anpassen der [Geometrie Toleranzen] können Sie die das Ergebnis der Feature-Erkennung für bauchige Flächen und Kanten und lückenhafte Flächenübergänge verbessern." %}

{% trans "Folgende Einstellparameter stehen zur Verfügung:" %}

 

 

{% trans "Position" %}

{% trans "[XY]-Positionsdaten deren Unterschied kleiner ist als dieser Wert, werden als identisch betrachtet. Und Abstände die kleiner sind als dieser Wert, werden als Null betrachtet." %}

 

{% trans "Kurven / Regelflächen-Linearität" %}

{% trans "Sind die Kantenübergänge von Fläche zu Fläche bauchiger (Kante ist nicht gerade sondern konvex od. konkav) als die hier eingestellte [Kurven / Regelflächen-Linearität], werden diese als nicht erodierbar eingestuft." %}

{% trans "Durch Erhöhen des Toleranzwertes für die [Kurven / Regelflächen-Linearität], kann man erreichen, dass auch bauchige Kantenübergänge als erodierbar eingestuft werden." %}

 

 

{% trans "Regelflächen-Toleranz" %}

{% trans "Wenn das System eine Regelfläche erzeugt, werden alle Geometrieelemente der oberen und unteren Regelflächengeometrie untersucht, ob man deren Geometrie vereinfachen kann." %}

{% trans "Dies wäre beispielsweise der Fall, wenn ein Kreis aus vielen kleinen Linienelementen bestünde, anstatt aus einem Bogenelement. Hier versucht System innerhalb des vom Parameter [Regelflächen-Toleranz] vorgegebenen Genauigkeit aus den Linienelemente einen oder mehrere Bogenelemente zu erzeugen." %}

{% trans "Der Tolerantwert bestimmt dabei die erlaubte Abweichung von der Originalgeometrie." %}

 

{% trans "Maximale Kantenabweichung" %}

{% trans "Im Bild unterhalb ist ein BSURFACE (Parametrische Fläche) zu sehen, deren interne Parameter-Linien (die [gelben] Linien) die [rot] gekennzeichnete Flächenkante überschneiden." %}

{% trans "Theoretisch würde der Schneiddraht diesen gelben Parameter-Linien folgen. Was jedoch ein unerwünschtes Schneidergebnis zur Folge hätte." %}

{% trans "Ändert man die Voreinstellung des Parameters [Maximale Kantenabweichung] von 5 Grad auf z.B. 25 Grad, würde dies in unserem Beispiel zu einem besseren Schneidergebnis führen." %}
