创建加工图形的示例

腐蚀 /2.5D 铣削/车削 模块中的几乎所有加工操作 都 基于 加工图形 几何类型。

在进一步阅读之前，熟悉基于 C 线、C 圆和线/弧元素创建设计非常重要。

为了能够进行编辑，必须将现有的线/弧元素组合成编辑图形。 这是按如下方式完成的：

  • 应根据以下几何形状创建加工图形（必须是直线/圆弧元素）。

  • 在准备 菜单中，选择 直线 /圆弧 => 处理图形 命令。

  • 在“起始孔” 字段中，输入 应穿线的位置或使用按钮 确定图形中的坐标 。

一个通知：

起始孔位置必须位于几何图形之前。 它可以位于 几何体的 内部 （模具）或 外部（印模）。

  • 单击 选择直线/圆弧 >> 按钮指定要使用哪些几何元素来创建加工图形。

  • 选择该按钮后，系统将显示 用于选择几何元素的 标准选择查询。

  • 选择 “连接” 选项（ 鼠标右键 或 水平图标栏 ）并单击几何 元素 。

  • 请注意系统如何自动选择以下所有几何元素。

  • 使用“结束” 选项结束选择 （请参见 鼠标右键 或 水平图标栏 ）。

  • 下图是成品图，包括起始孔位置。

  • 创建其他编辑图形或使用“确定” 按钮关闭对话框 。
