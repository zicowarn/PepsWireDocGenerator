手动面部选择

当需要编辑 几何图形的部分区域 并且自动特征识别方法会检测到太多或不需要的区域时，可以使用此方法。

如果选择此选项，则 依次 单击要编辑的 所有 面。 请注意，所选面必须相互连接。

方法：

  • 依次 单击 要编辑的几何体的所有面。 按住Ctrl 键的同时选择 。 WIRE-Expert 检查所选面是否可腐蚀。

  • 用 结束选择 。

  • 使用 确认特征创建 。

  • 依次 单击所有面 ，即 要编辑的下一个几何体 。

  • 也以 结束下一个选择 。

  • 使用 确认特征创建 。

  • 用 结束对话框 。
