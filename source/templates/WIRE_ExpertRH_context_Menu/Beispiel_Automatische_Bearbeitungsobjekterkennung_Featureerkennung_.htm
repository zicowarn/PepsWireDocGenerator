自动特征检测

此示例旨在展示如何使用 自动特征识别 自动创建剪切编辑。

右键 单击 功能管理器 中的“ Operation1 ” 分支。

将打开上下文菜单。 选择 自动 特征检测 命令。

将显示“特征检测” 对话框 。 在对话框中，您可以指定自动识别例程应识别哪些对象。

在以下示例中，不会编辑模型的外部轮廓（仅 内部特征 (die )）。

要设置特征检测，请单击 特征检测设置 按钮。

这将打开一个对话框，您可以在其中设置功能检测。

确保将相同的功能分组为一项功能，并启用 “分组相同的功能” 。

使用 来应用设置 。

还可以使用 关闭 “ 特征检测” 对话框 。

系统在实体中搜索所有易腐蚀的几何形状（特征识别），并根据预先设置的切削参数生成加工建议。

系统在实体中搜索所有易腐蚀的几何形状（特征检测）。

对于每个可腐蚀几何体，系统会 在 FeatureManager 设计树中创建一个 单独的特征 ：

现在根据特征创建 加工操作 和 图形 。

右键 单击 “特征管理器” 中的 分支 ，然后从上下文菜单中选择“创建图形 + 特征 编辑” 。

此功能创建 加工图形 （2D 加工几何形状）以及 加工操作 并将它们传输到 操作列表 。

注意： 您还可以选择 单独的 功能。
