通过表面颜色进行特征检测

在此示例中，自动特征识别应 识别模型中的 所有浅绿色区域并仅处理它们。 （您正在寻找的突破点在下图中用箭头符号 标记 ）。

创建颜色搜索列表

右键 单击 OPTICAM FeatureManager 中的 “Operation1” 分支 ，然后选择 “自动 特征检测”命令

这将打开 “自动特征检测” 对话框。

首先，我们创建一个 颜色搜索列表 ，在其中确定要识别的颜色。

要创建 颜色搜索列表， 请选择对话框中的按钮 。

将显示“颜色设置” 对话框 。

在列表名称 字段中，输入 颜色列表的名称（ 浅绿色 ）。

然后单击 创建新列表 按钮。

现在单击 模型中的 浅绿色区域 。

所需区域的颜色将传递到对话框并输入到 “列表中的颜色” 字段中。


注意 ：可以选择具有不同颜色的其他区域来 创建具有多种颜色的 颜色搜索列表。

使用 应用颜色设置 。

按颜色执行自动特征检测

系统跳回自动特征识别对话框。

在颜色列表 对话框区域中，选择 您刚刚创建的 浅绿色 列表 。

启用 “使用颜色列表” 选项以使系统仅搜索彩色区域（在本例中仅搜索 浅绿色 区域）。

确保对话框填写如下：

使用 来应用设置 。

自动特征识别功能会在实体中搜索 所有可腐蚀的浅绿色几何形状 ，并根据预先设置的切削参数生成加工建议。
