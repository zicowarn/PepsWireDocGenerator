机械窗口

 

如何创建新机器

有一组不同的文件是正常运行机器所必需的。

 

机器模型文件 (STL)

STL 文件定义机器的几何组件。 STL 文件的名称在 xml 中用于将它们分配给适当的轴。 通常您有多个 STL 文件，因为您的机器由多个轴组成。

这里重要的是机器几何形状与绝对中心点 (0, 0, 0) 对齐。 这意味着绝对中心点位于机器的刀具固定架中。 此时将放置刀架。 通常（但不是必须）该点应与工作台的顶面中心点相对应。 换句话说，刀具固定架和工作台顶面中心点应接触在同一点，即创建 STL 文件的 CAD 软件的 0,0,0 点。 其余机器部件也应进一步参考这一点。

因此，将机器和主轴移动到 CAD 模型中的正确位置。 此后，您可以简单地将单个零件保存到 STL 文件。 那么他们已经有了正确的位置。

用于本教程的机器已经以正确的方式对齐。

 

反射纹理 (BMP)

这用于反射的纹理。 请注意，仅允许使用 .bmp 文件格式（不允许使用 .jpg、.gif 等）。

以下教程描述了创建机器的必要步骤。 必要的机器模型文件（STL 文件）和反射纹理（tablereflection.bmp）可以在 new_machine.zip 存档中找到。

 

创建新机器

将在运动学树窗口中创建一个新机器。 因此，如果您已加载机器模拟，请切换到机器窗口。 您通常已经选择了一台机器来进入机器模拟。 该机器已列在树中。

要创建新机器，请点击图标栏中的“新机器”按钮。

现有计算机将被删除，并出现一个新的、空的未命名计算机。 现在启用机器编辑（如果尚未选择）。 这将允许您修改树中的项目。

单击“未命名机器”。 名称选项出现在窗口底部。

使用“new_machine”更改计算机的名称。 该名称是您的新机器的名称。

请记住：机器 ID 名称和机器数据的文件夹名称必须相同。 在这种情况下，我们将机器称为“new_machine”。 点击“将机器另存为..”按钮。 将“new_machine.xml”保存到“new_machine”文件夹。

 

安装外壳

外壳被视为机器的固定部分。 固定意味着外壳不会移动并保持静止，因为它没有连接到线性或旋转轴。

转到机器窗口。 点击“编辑机器”按钮即可更改为编辑模式。

首先要做的就是为您的机器命名。

Right click the 'new_machine' in the kinematics tree and choose 'Add Geometry' from the drop down window. Select the 'HOUSING.STL' from the 'new_machine'directory.

There is a special setting about the housing. When you are running machine simulation you have the opportunity to blank out the machine housing by pushing that button.

These blank able parts will be defined when you add to your name the prefix 'mh_'. So, all parts with the prefix mh_ will be blanked out. In our case the name would be 'mh_housing'.

Properties for new geometry are displayed in bottom of window. Change the ID with "mh_housing" and also you can set up the color. Pick any colour.

Save the machine. This time use the 'Save Machine' button. That simply saves the changes without bringing the file browser.

 

Attaching the kinematic axes

As like the housing, the axes will be added into the kinematic tree and its subbranches. The structure of the machine parts is hierarchically. The dependence of an tree item always start from the top item of a branch. The main branch itself is the machine top item which has the housing attached.

To give an example we look at our 'new_machine'. The 'new_machine' is a so called 'table table' machine. That means that both rotation axes (A and C) lie in the table of the machine. The A axis (blue) and C axis (purple) axis are attached to the Y axis (grey). The table is independent from the linear axes X (green) and Z (orange). Accordingly the Y axis and the X axis build a new branch for themselves.

To add a kinematic axis you have to do 2 things:

Add the kinematic description

Attach the machine model part

 

Adding the kinematic description

Right click on the 'new_machine', go to 'add Transl. Axis' and select 'Y Axis'. The property window opens automatically.

The new Y axis is added. When you select it from the tree, the properties menu appears in bottom of window. The properties for 'ID', 'Direction', 'Limits' are setup by default.

The first thing you can change here is ID name. Due to its definition the kinematic axes ID's can't be named randomly (like 'Y axis').

 

The next thing to be changed is the kinematic orientation, which is the direction the linear axis should move. Here the direction must be expressed by a vector which is described in the coordinates X Y and Z. To let the linear axis move in Y, setup value 1 for Y.

Last option that can be changed is the limit for that axis. Setup minimum limit -80 and maximum limit 100.

The Y axis appeared in a new branch underneath the machine.

The next thing to do is to attach the A axis onto the Y axis. Right click on the Y axis, go to 'Add rotation axis' and select 'A Axis'.

Properties menu from the bottom of window are setup by default.

Center point must be changed from default value. This center point defines the position where this axis rotates around. Due to its definition you have give that information to the kinematic. The distance will be measured in X, Y and Z to the absolute center point. The A axis is shifted, so you have to add the shift distance accordingly. In our case it is -106 mm in Z. Our direction is Y 1, X and Z 0 (it's turning around Y axis).

The limits now will be expressed by angles since it is a rotation axis. Enter the range from -90 to 90.

Attaching the machine model part

Right click the desired kinematic axis in the kinematic tree. Then choose 'Add Geometry' from the drop down window. Enter a random ID to the geometry and select the geometry file. The files can be found in the 'new_machine.zip', so make sure you take the right geometry. Attach following geometries:

 

A axis (tilt axis), 'A_AXIS.STL'.

C axis (rotation axis), 'C_AXIS.STL'.

X axis (linear axis), 'X_AXIS.STL'.

Y axis (linear axis), 'Y_AXIS.STL'.

Z axis (linear axis), 'Z_AXIS.STL'.

 

 

Attaching the workpiece set

The workpiece, toolpath, stock and initial stock are so called dynamic elements. That means that they can be changed during simulation and don't stay static like the axes geometries.

In order to attach the elements onto the table, you have to define a certain coordination transformation.

Now right click on the C axis, select branch 'Add Dynamic Element' and add workpiece set. This option added a group of options : workpiece, stock, initial stock and toolpath. Also you can add those options manually.

 

Attaching the tool

The tool also is a dynamic element. It changes during simulation and doesn't stay static like the axes geometries.

In order to attach the spindle, you have to define a certain coordination transformation.

Right click on the Z axis (which in this case is the spindle), select branch 'Add Dynamic Element' and add 'Tool Set'.

 

Defining the collision pairs

For the machine simulation it is very important that collisions will be recognized. Therefore you can set collision control sets for machine parts.

这里重要的是找到单个部分之间的正确依赖关系。 例如。 在 Y 轴和外壳之间进行碰撞检查是没有意义的。 Y 轴永远不会损坏外壳，因为它在顶部移动。 重要的是平移轴和旋转轴之间、工件与旋转和平移轴之间的碰撞检查，当然还有工具和工件之间的碰撞检查

 

右键单击“new_machine”计算机并选择“添加 collcheck”。

 

选择 CollisionCheck 后，属性菜单将出现在窗口底部。

您可以通过设置两个组来定义碰撞检查。 将检查两组之间的碰撞。

因此，在左侧组中，您可以添加工件及其所属的所有几何形状，在右侧组中，添加工具以及所有几何形状。

 

碰撞检查集将添加到机器树中。

选择 CollisionCheck 后，属性菜单将出现在窗口底部。

单击“+”或“x”按钮来添加或删除对象。
