Punkte

{% trans "Die [Punkt]-Funktion liefert Information über die aktuell angeklickte (! [mittlere] Maustaste verwenden) Koordinate. Es wird die XYZ-Koordinate, das verwendete Werkzeug und das Materialaufmaß (engl. Deviation) angezeigt." %}

{% trans "Beispiel:" %}

{% trans "Bewegen Sie den Cursor auf die gewünschte Position. Klicken Sie mit der [mittleren] Maustaste. Es wird eine kleine Infobox mit Informationen über den angeklickten Punkt gezeigt. Die Informationen werden zusätzlich im [Messen]-Dialog in einer Liste notiert." %}

{% trans "Folgende Informationen werden angezeigt:" %}

{% trans "[Position]: Sind die X-, Y- und Z-Koordinaten des angeklickten Punktes." %}

{% trans "[Abweichung (Deviation)]: Zeigt die Dicke des Aufmaßes, das heißt die Abweichung des Restmaterials vom [Zielteil (Werkstück)]. In unserem Beispiel beträgt das gemessene Aufmaß dies 0.0015 mm" %}

{% trans "[Wkz. Nummer (Tool number)]: Zeigt die Nummer des verwendeten Werkzeugs." %}

{% trans "[Satznummer (Move number)]: Satznummer in der Liste der NC-Sätze." %}

{% trans "[Ziel ID (Target ID)]: Zeigt die interne Nummer der Zielteilgeometrie" %}

{% trans "[Hinweis]: Um das Restmaterial farblich zu kennzeichnen, müssen Sie in den Dialog #[Rohteilanalyse] / [Abweichung]#[ ]wechseln." %}

{% trans "[[Abweichung]]: Ist die Abweichung des Restmaterials vom Zielteil. In dem Beispielbild oberhalb beträgt die Abweichung 0.0015 mm" %}

Bereich vergrößern

{% trans "Hier können Sie einen Teilbereich des Rohteils mittels eines rechteckigen Auswahlfensters auszuwählen und auszuschneiden. Das Material [außerhalb] des Fensters wird entfernt." %}

{% trans "Der Bereich [innerhalb] des Fensters wird neu berechnet und mit einer höheren Darstellungs-Genauigkeit angezeigt." %}

{% trans "Beispiel:" %}

{% trans "Bewegen Sie den Cursor auf die gewünschte Position. Klicken Sie mit der [mittleren] Maustaste, um den [oberen linken Punkt] des Auswahlfensters zu definieren. Klicken Sie erneut die [mittlere] Maustaste, um den [unteren rechten Punkt] des Auswahlfensters zu definieren." %}

{% trans "Klicken Sie im Dialog auf die [Dynamisch vergrößern] @ Schaltfläche, um den Prozess zu starten." %}

{% trans "Klicken Sie im Dialog auf die [Dynamisch vergrößern zurücksetzen] @ Schaltfläche, um den Prozess zu beenden." %}

 

 

Lösche Teil (Auswahl)

{% trans "Mit dieser Funktion kann der Benutzer interaktiv die Ausfallteile löschen, die losgelöst vom Werkstück angezeigt werden." %}

{% trans "Klicken Sie einfach das zu löschende Ausfallteil an (! [mittlere] Maustaste verwenden)." %}

{% trans "Beachten Sie bitte, dass das System die Ausfallteile nur dann in unterschiedlichen Farben anzeigt, wenn im Vorfeld die Rohteilanalysefunktion [#Teile markieren#] aktiviert wurde." %}

 

{% trans "[Beispiel]: Das folgende Bild zeigt links ein gefrästes Werkstück mit mehreren Ausfallteilen. Das violette Ausfallteil wurde mit der[ mittleren Maustaste] angeklickt, um es zu löschen. Das rechte Bild zeigt das Ergebnis." %}

 

 

Lösche alle Teile (behalte nur 1)

{% trans "Diese Funktion löscht [alle] vorhandenen Ausfallteile [außer] dem aktuell, mit der ! [mittleren] Maustaste ausgewähltem Ausfallteil." %}

{% trans "Beachten Sie bitte, dass das System die Ausfallteile nur dann in unterschiedlichen Farben anzeigt, wenn im Vorfeld die Rohteilanalysefunktion [#Teile markieren#] aktiviert wurde." %}

 

{% trans "[Beispiel]: Das folgende Bild zeigt links ein gefrästes Werkstück mit mehreren Ausfallteilen. Das orangefarbene Werkstück Ausfallteil wurde mit der[ mittleren Maustaste] angeklickt, um es zu behalten und alle anderen Ausfallteile zu entfernen. Das rechte Bild zeigt das Ergebnis." %}

 

 

Lösche Teile (automatisch)

{% trans "Dieser Funktion behält alle Ausfallteile, die nach einer Simulation mit der ! [mittleren] Maustaste ausgewählt wurden." %}

{% trans "Klickt man die Ausfallteile an, wird die Auswahl durch ein kleines 'Fähnchen' angezeigt (Siehe Beispiel unterhalb) . Klicken Sie anschließend auf die Schaltfläche [Anwenden], um die Auswahl auszuführen." %}

{% trans "Dies führt dazu, dass nur die ausgewählten Ausfallteile übrig bleiben." %}

{% trans "[Beispiel]: Das folgende Bild zeigt links ein gefrästes Werkstück mit mehreren Ausfallteilen. Es wurden 3 Ausfallteile mit der[ mittleren Maustaste] angeklickt, die übrig bleiben sollen, der Rest wird automatisch gelöscht, wenn man die Schaltfläche [Anwenden ]klickt. Das rechte Bild zeigt das Ergebnis." %}

{% trans "Mit dieser Funktion kann der Anwender ausgewählte Ausfallteile vom Löschvorgang ausnehmen. Die Auswahl findet statt, indem man auf jedem [nicht] zu löschenden Ausfallteil eine Punktposition mit der [mittleren Maustaste ]anklickt. Alle anderen Ausfallteile werden gelöscht." %}

{% trans "Die Ausfallteile werden gelöscht, wenn sie während der Simulation entstehen (wenn man vorher die Punktpositionen gewählt halt) oder jedes mal, wenn man auf die Schaltfläche [Anwenden] klickt." %}

{% trans "Beachten Sie bitte, dass das System die Ausfallteile nur dann in unterschiedlichen Farben anzeigt, wenn im Vorfeld die Rohteilanalysefunktion [#Teile markieren#] aktiviert wurde." %}

 

 

Entferne Elemente (mit Test)

{% trans "Mit dieser Funktion können Sie testen, ob sich Ausfallteile überhaupt entnehmen lassen. Es existieren Situationen, in denen sich beispielsweise beim [Drahterodieren] eines konischen Durchbruchs, dass sich das Ausfallteil nicht oder nur in eine bestimmte Richtung entnehmen lässt. Das folgende Beispiel soll so eine Situation zeigen:" %}

{% trans "[Beispiel]: Das folgende Bild zeigt einen konischen Durchbruch und das Ergebnis eines Tests auf Ausfallteile. Das Ergebnis des Tests zeigt, dass das Ausfallteil (grüner Volumenkörper) nur nach unten in Richtung (Z-) entnommen werden kann. Ein Herausziehen des Ausfallteils nach oben (Z+) ist nicht möglich." %}

 

{% trans "[Vorgehensweise]:" %}

{% trans "Alle Ausfallteile werden im Zuge der Simulation in dem Dialog aufgelistet." %}

{% trans "Um den Test für ein Ausfallteil durchzuführen, aktiviert man das Ausfallteil in der Liste der Ausfallteile und drückt die Schaltfläche [@Test]. Dies führt den Test aus." %}

 

{% trans "Schaltflächen" %}

{% trans "[@ Test]:" %}

{% trans "Diese Funktion verschiebt die gewählten Ausfallteile in die Achsenrichtung, die im unteren Dialogbereich [Zum Testen wähle Richtung] eingestellt wurde." %}

{% trans "Die Funktion [Test] betrachtet auch den eingestellten [Offset Winkel ]der Richtungsachse und um die Richtungsachse herum." %}

{% trans "[@ Entferne ausgewähltes Bauteil]" %}

{% trans "Diese Schaltfläche löscht alle in der Liste gewählten Ausfallteile." %}

{% trans "@ Stop wenn neuer Chip (Ausfallteil) erkannt wird" %}

{% trans "Ist diese Schaltfläche gewählt, erscheint jedes mal ein Hinweisfenster, wenn das System ein Ausfallteil entdeckt." %}

{% trans "Die Dialogparameter '[Von Richtungsachse]' und '[Um Richtungsachse]' ermöglichen es, Ausfallteiltests in jeder beliebigen Richtung auszuführen." %}

{% trans "Die beiden oberhalb genannten Parameter, erweitern das Spektrum der erlaubten Richtungen für Ausfallteiltests, weil man Tests in einer offsettierten Richtung zur Hauptachse oder in einer offsettierten Richtung um die Hauptachse herum ausführen kann. Die Offsetwerte müssen durch den Anwender bestimmt werden." %}
