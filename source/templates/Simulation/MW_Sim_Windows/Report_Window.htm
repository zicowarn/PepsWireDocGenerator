报告窗口

 

报告窗口收集模拟期间发生的所有特殊事件（例如碰撞）。 该报告以树形结构的形式列出。

树结构的每个分支都以操作编号开始，后面跟着工具编号。 典型事件有：

价值超出

低估

轴几何形状之间的碰撞事件（例如 C 轴和工作台之间的碰撞）

刀具和几何形状之间的碰撞事件（例如刀具和 C 轴之间的碰撞）

刀具和几何形状之间的碰撞事件（例如刀具和 C 轴之间的碰撞）

刀具与工件之间的碰撞事件

毛坯和刀具部分之间的碰撞事件（例如毛坯和刀架之间的碰撞）

快速移动过程中毛坯和刀具路径之间的碰撞事件

刀具与毛坯或工件之间的接近报警（接近碰撞）。

轴几何形状之间的接近警报（接近碰撞）。

 

 

主要功能：

当工具执行碰撞时，碰撞深度可用（XXX 例如范围：大于 0.2）。

使用 “显示字段选择”对话框自定义 “报告” 窗口中显示的列 。 为此， 请右键 单击 报告 窗口，然后从上下文菜单中选择 “显示字段选择” 。

类型 – 显示事件类型，以图标表示。

Tool# - 显示此事件中使用的 工具编号 。

操作编号 - 显示此事件中使用的 操作编号 。

块 – 显示当前事件在记录列表中的记录编号。

注释 ——显示当前事件的操作注释。

 

注意 ：如果单击对话框中的字段列（例如 ID#），则可以对该列进行排序。

 

如果单击对话框中的某个事件，工具将跳转到工具路径中的相应坐标。
